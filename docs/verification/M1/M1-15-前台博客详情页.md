# M1-15 - 前台博客详情页

> 对应任务: [01-迭代计划.md](../../plan/01-迭代计划.md#m1-15-前台博客详情页markdown渲染-目录-相关文章)
> 涉及 Spec: [09-前端路由设计.md](../../spec/09-前端路由设计.md)

## 验证目标

前台博客详情页 Markdown 渲染正常，显示目录和相关文章。

## 测试用例

### TC-01: 详情页访问

**前置条件：**
- 存在已发布的文章

**测试步骤：**
1. 访问 `/blog/{slug}`
2. 验证页面加载

**预期结果：**
- 页面正常显示
- SEO meta 标签正确

### TC-02: Markdown 渲染

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 查看文章内容区域
2. 验证 Markdown 渲染

**预期结果：**
- 标题、段落、列表、代码块等正确渲染
- 语法高亮正常（代码块）

### TC-03: 文章目录

**前置条件：**
- 文章包含多个标题

**测试步骤：**
1. 查看文章右侧或顶部目录
2. 验证目录显示

**预期结果：**
- 目录列出文章标题
- 点击跳转正确

### TC-04: 相关文章

**前置条件：**
- 存在同分类的其他文章

**测试步骤：**
1. 查看文章底部相关文章
2. 验证显示

**预期结果：**
- 显示 2-4 篇相关文章
- 点击跳转正确

### TC-05: 分类标签显示

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 查看文章元信息区域
2. 验证分类和标签显示

**预期结果：**
- 显示所属分类（可点击）
- 显示标签列表（可点击）

### TC-06: 返回列表

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 点击"返回列表"按钮
2. 验证跳转

**预期结果：**
- 跳转到 `/blog` 列表页

- [ ] TC-03: 文章目录通过

- [ ] TC-06: 返回列表通过
