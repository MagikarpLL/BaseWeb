## 12.4.2 具体规格

### 12.4.2.1 推荐算法

```java
// 伪代码实现
public List<ScoredPost> calculateRelatedPosts(BlogPost current, List<BlogPost> candidates) {
    Map<Long, Double> scores = new HashMap<>();

    for (BlogPost post : candidates) {
        double score = 0.0;

        // 相同分类 +3 分
        if (post.getCategoryId().equals(current.getCategoryId())) {
            score += 3.0;
        }

        // 相同标签 +2 分/标签
        Set<Long> currentTags = new HashSet<>(current.getTagIds());
        Set<Long> postTags = new HashSet<>(post.getTagIds());
        int commonTags = Sets.intersection(currentTags, postTags).size();
        score += commonTags * 2.0;

        // 发布时间相近（7天内）+1 分
        if (isWithinDays(post.getPublishedAt(), current.getPublishedAt(), 7)) {
            score += 1.0;
        }

        // 阅读量较高 +0.5 分
        if (post.getReadCount() > averageReadCount) {
            score += 0.5;
        }

        scores.put(post.getId(), score);
    }

    // 按评分排序，取 TOP 3
    return candidates.stream()
        .sorted((a, b) -> Double.compare(scores.get(b.getId()), scores.get(a.getId())))
        .limit(3)
        .collect(toList());
}
```

### 12.4.2.4 Mapper SQL

```xml
<select id="findRelatedPosts" resultType="BlogPost">
    SELECT p.* FROM blog_post p
    WHERE p.status = 'published'
      AND p.id != #{excludeId}
    <if test="categoryId != null">
      AND p.category_id = #{categoryId}
    </if>
    ORDER BY
      CASE WHEN p.category_id = #{categoryId} THEN 3 ELSE 0 END +
      p.read_count * 0.01 +
      CASE WHEN p.published_at > NOW() - INTERVAL '7 days' THEN 1 ELSE 0 END
    DESC
    LIMIT #{limit}
</select>
```
