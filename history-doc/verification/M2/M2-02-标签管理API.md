# M2-02 - 标签管理 API

> 对应任务: [01-迭代计划.md](../../plan/01-迭代计划.md#m2-02-标签管理-api)
> 涉及 Spec: [02-API契约.md](../../spec/02-API契约.md)

## 验证目标

管理员可增删改查标签。

## 测试用例

### TC-01: 创建标签

**前置条件：**
- 管理员已登录

**测试步骤：**
1. POST `/api/admin/tags` 请求体 `{"name": "Vue3", "slug": "vue3"}`
2. 验证返回 200 和标签数据

**预期结果：**
- 返回标签 ID 和创建时间
- 数据库中标签已创建

### TC-02: 标签列表

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. GET `/api/admin/tags`
2. 验证返回包含刚创建的标签

**预期结果：**
- 列表包含 name、slug 字段

### TC-03: 更新标签

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. PUT `/api/admin/tags/{id}` 请求体 `{"name": "Vue 3.0"}`
2. 验证返回更新后的数据

**预期结果：**
- 名称已更新

### TC-04: 删除标签

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. DELETE `/api/admin/tags/{id}`
2. 验证返回 204

**预期结果：**
- 删除成功
- 数据库中标签已软删除
