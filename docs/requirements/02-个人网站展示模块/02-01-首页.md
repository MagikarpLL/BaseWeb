# 02-01-首页

## 2.1.1 功能概述

首页是网站的入口页面，承担以下职责：
- **第一印象**：给访客展示个人品牌和专业能力
- **导航入口**：引导访客进入其他页面（博客、工具、关于）
- **快速预览**：展示最新文章和精选工具

## 2.1.3 功能详情

### Hero 区域

| 字段 | 说明 | 数据来源 |
|------|------|----------|
| 头像 | 用户头像，圆形，150x150px | 后台配置 |
| 姓名 | 显示姓名 | 后台配置 |
| 职位/头衔 | 一句话描述身份 | 后台配置 |
| 个人简介 | 1-2 句话介绍 | 后台配置 |
| 社交链接 | GitHub, Email, Twitter 等 | 后台配置 |

### 最新文章区域

| 字段 | 说明 |
|------|------|
| 标题 | 文章标题，最多 50 字符 |
| 摘要 | 文章摘要，最多 150 字符 |
| 发布日期 | 格式：YYYY-MM-DD |
| 阅读量 | 显示数字 |
| 链接 | 点击跳转至文章详情页 |

**显示规则**：取最近发布的 3 篇已发布文章，按发布时间倒序。

### 精选工具区域

| 字段 | 说明 |
|------|------|
| 工具名称 | 最多 20 字符 |
| 工具图标 | Font Awesome 或自定义 SVG |
| 工具描述 | 最多 80 字符 |
| 链接 | 点击跳转至工具详情页 |

**显示规则**：取后台标记为"精选"的 3 个工具。

## 2.1.4 SEO 元数据

| 标签 | 内容 |
|------|------|
| title | `{姓名} - 个人网站` |
| description | `{姓名}，{职位}的个人网站，提供博客和技术分享` |
| og:title | `{姓名} - 个人网站` |
| og:description | `{职位}的个人网站` |
| og:type | website |
| og:url | https://yoursite.com/ |

## 2.1.5 用户交互

| 交互 | 行为 |
|------|------|
| 点击文章卡片 | 跳转至 `/blog/{slug}` |
| 点击工具卡片 | 跳转至 `/tools/{toolPath}` |
| 点击"查看更多文章" | 跳转至 `/blog` |
| 点击社交链接 | 新窗口打开对应链接 |
