# 10-01-管理后台

> 对应 Requirement: [10-01-管理后台.md](../requirements/10-管理后台模块/10-01-管理后台.md)

## 10.1.1 设计原则

| 原则 | 说明 |
|------|------|
| 统一布局 | 所有管理页面共享侧边栏布局 |
| 高效导航 | 侧边栏支持折叠，快速定位 |
| 操作一致 | 全局统一的保存/取消/删除交互 |

---

## 10.1.2 布局规格

### 整体布局

```
┌─────────────────────────────────────────────────────────────┐
│  顶部栏: Logo  ·  网站名称  ·            管理员  [退出]     │
├──────────┬──────────────────────────────────────────────────┤
│          │                                                   │
│  侧边栏  │              主内容区                              │
│  240px  │                                                   │
│          │                                                   │
└──────────┴──────────────────────────────────────────────────┘
```

### 顶部栏

| 元素 | 规格 |
|------|------|
| Logo | 左侧，宽 `40px`，点击跳转 `/` |
| 网站名称 | 来自 siteName 配置 |
| 管理员 | 右侧，显示用户名 |
| 退出按钮 | 右侧，点击执行登出 |

### 侧边栏

| 元素 | 规格 |
|------|------|
| 宽度 | 240px（桌面）/ 64px（折叠） |
| 背景 | `#f8f9fa` |
| 导航项 | 图标 + 文字，高度 `48px` |
| 高亮 | 当前页导航项背景 `#e8f0fe`，左侧 3px 边框 `#667eea` |
| 分组 | 用分隔线分组（仪表盘 / 文章管理 / 分类标签 / 评论 / 配置 / 统计） |

**导航项**：
| 编号 | 导航 | 图标 | 路由 |
|------|------|------|------|
| 1 | 仪表盘 | `fa-solid fa-gauge` | `/admin` |
| 2 | 文章管理 | `fa-solid fa-file-lines` | `/admin/posts` |
| 3 | 新建文章 | `fa-solid fa-plus` | `/admin/posts/new` |
| 4 | 分类管理 | `fa-solid fa-folder` | `/admin/categories` |
| 5 | 标签管理 | `fa-solid fa-tag` | `/admin/tags` |
| 6 | 评论管理 | `fa-solid fa-comment` | `/admin/comments` |
| 7 | 网站配置 | `fa-solid fa-gear` | `/admin/settings` |
| 8 | 数据统计 | `fa-solid fa-chart-line` | `/admin/analytics` |

## 10.1.3 仪表盘页面规格

### 统计卡片

| 卡片 | 数据来源 |
|------|----------|
| 文章总数 | `SELECT COUNT(*) FROM blog_post WHERE status='published'` |
| 草稿数 | `SELECT COUNT(*) FROM blog_post WHERE status='draft'` |
| 待审评论 | `SELECT COUNT(*) FROM comment WHERE status='pending'` |
| 今日 PV | 今日 page_view 表 pv_count 总和 |

**卡片样式**：
- 宽：`25%`（桌面）/ `50%`（平板）
- 高：`100px`
- 数字：字号 `2rem`，加粗
- 标签：字号 `0.875rem`，颜色 `#666`
- 点击跳转对应管理页

### 最近文章

| 元素 | 规格 |
|------|------|
| 标题 | 最近 5 篇 |
| 内容 | 文章标题 + 状态标签 + 日期 |
| 操作 | 点击跳转编辑页 |

### 待审核评论

| 元素 | 规格 |
|------|------|
| 标题 | 最近 5 条待审评论 |
| 内容 | 评论者昵称 + 评论摘要（20字）+ 来源文章 |
| 操作 | 点击跳转审核页面 |

## 10.1.4 全局交互规则

### 表单操作

| 操作 | 行为 |
|------|------|
| 保存 | 成功后 Toast "保存成功"，停留当前页 |
| 取消 | 弹出确认"是否放弃编辑？"，确认后返回列表页 |
| 删除 | 弹出确认"确认删除？此操作不可恢复"，确认后删除并刷新列表 |

### 列表页规则

| 规则 | 说明 |
|------|------|
| 默认分页 | 每页 10 条 |
| 空状态 | 显示"暂无数据"插图 + 引导文案 |
| 加载状态 | 显示骨架屏 |
| 删除后 | 自动刷新列表，若当前页无数据则回到上一页 |

---

## 10.1.5 响应式适配

| 断点 | 行为 |
|------|------|
| >= 1024px | 侧边栏常驻展开（240px） |
| 768-1023px | 侧边栏折叠为图标（64px），hover 展开 |
| < 768px | 侧边栏隐藏，汉堡菜单触发侧滑抽屉 |

**汉堡菜单**：
- 位置：顶部栏左侧
- 图标：`fa-solid fa-bars`
- 侧滑：从左滑出，宽度 `280px`

---

## 10.1.6 权限控制

| 规则 | 说明 |
|------|------|
| JWT 验证 | 所有 `/admin/*` 需验证有效 Token |
| 未登录处理 | 重定向到 `/login?redirect=当前路径` |
| Token 过期 | 自动刷新，失败则跳转登录 |
| 审计日志 | 增删改操作记录（后续迭代） |
