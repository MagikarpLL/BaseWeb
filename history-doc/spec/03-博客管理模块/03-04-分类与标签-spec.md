# 03-04-分类与标签

> 对应 Requirement: [03-04-分类与标签.md](../requirements/03-博客管理模块/03-04-分类与标签.md)

## 3.4.1 设计原则

| 原则 | 说明 |
|------|------|
| 简洁直观 | 管理界面清晰，操作简单 |
| 安全删除 | 删除分类需处理关联文章，防止孤儿数据 |
| 自动生成 | slug 自动从名称生成，保证 URL 友好 |

## 3.4.2 数据模型

### 分类 API

**接口**: `GET /api/admin/categories`

```json
{
  "code": 200,
  "data": [
    { "id": 1, "name": "前端", "slug": "frontend", "description": "前端技术相关文章", "sort": 1, "postCount": 20 }
  ]
}
```

**接口**: `POST /api/admin/categories` (创建)

```json
{
  "name": "前端",
  "slug": "frontend",
  "description": "前端技术相关文章",
  "sort": 1
}
```

**接口**: `PUT /api/admin/categories/:id` (更新)

**接口**: `DELETE /api/admin/categories/:id` (删除)

### 标签 API

**接口**: `GET /api/admin/tags`

```json
{
  "code": 200,
  "data": [
    { "id": 1, "name": "Vue.js", "slug": "vuejs", "postCount": 10 }
  ]
}
```

**接口**: `POST /api/admin/tags` (创建)

```json
{
  "name": "Vue.js",
  "slug": "vuejs"
}
```

**接口**: `PUT /api/admin/tags/:id` (更新)

**接口**: `DELETE /api/admin/tags/:id` (删除)

## 3.4.3 页面渲染规格

### 分类管理页

| 元素 | 渲染规则 |
|------|----------|
| 列表布局 | 表格行：名称 / slug / 描述 / 文章数 / 操作 |
| 操作 | 编辑 / 删除 按钮 |
| 排序 | 可拖拽调整顺序 |
| 新建按钮 | 右上角，`+ 新建分类` |

**表格列**：
| 列 | 宽度 | 说明 |
|---|---|---|
| 名称 | 120px | 分类名称 |
| slug | 150px | URL 标识 |
| 描述 | 200px | 分类描述（最多 200 字符） |
| 文章数 | 80px | 该分类下的文章数量 |
| 操作 | 120px | 编辑/删除 |

### 标签管理页

| 元素 | 渲染规则 |
|------|----------|
| 标签云布局 | 卡片网格，每卡片显示名称 + 文章数 |
| 卡片操作 | hover 显示编辑/删除按钮 |
| 新建按钮 | 右上角，`+ 新建标签` |

**标签卡片样式**：
- 宽度：`120px`，高度：`80px`
- 背景：`#f8f9fa`，圆角 `8px`
- hover：阴影加深

### 分类编辑 Modal

| 字段 | 规则 |
|------|------|
| 名称 | 必填，最多 20 字符 |
| slug | 自动生成（可编辑），小写字母/数字/连字符 |
| 描述 | 可选，最多 200 字符 |
| 排序 | 数字，值越小越靠前 |

### 标签编辑 Modal

| 字段 | 规则 |
|------|------|
| 名称 | 必填，最多 15 字符 |
| slug | 自动生成（可编辑） |

## 3.4.4 关键逻辑

### slug 自动生成规则

```typescript
function generateSlug(name: string): string {
  return name
    .toLowerCase()
    .replace(/[^\w\u4e00-\u9fa5]/g, '')  // 移除非字母数字和中文
    .replace(/\s+/g, '-')                 // 空格替换为连字符
    .replace(/-+/g, '-')                 // 多个连字符合并
}
```

### 分类删除处理

| 关联文章数 | 删除策略 |
|------------|----------|
| 0 | 直接删除 |
| > 0 | 弹窗选择：转移文章到其他分类，或强制删除文章 |

**删除确认弹窗**：
```
确认删除分类「前端」？

该分类下有 20 篇文章，请选择处理方式：
○ 转移文章到 [下拉选择目标分类]
○ 同时删除文章（不可恢复）

[取消] [确认删除]
```

### 标签删除处理

- 直接删除，自动解除与文章的关联
- 无需处理关联文章
