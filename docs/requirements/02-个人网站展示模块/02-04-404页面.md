# 02-04-404 页面

## 2.4.1 功能概述

404 页面在用户访问不存在的路由时显示，提供友好的错误提示和导航选项。

## 2.4.2 页面设计

```
┌─────────────────────────────────────────────────────────────┐
│                        Header                                │
│                   (简化版，不显示导航)                         │
└─────────────────────────────────────────────────────────────┘
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│                                                              │
│                       404                                    │
│                    大号醒目文字                               │
│                                                              │
│                  "页面不存在"                                 │
│                 "您访问的页面已失效或不存在"                    │
│                                                              │
│              ┌─────────────────────┐                         │
│              │    返回首页         │                         │
│              └─────────────────────┘                         │
│              ┌─────────────────────┐                         │
│              │    返回博客       │                         │
│              └─────────────────────┘                        │
│                                                              │
│                                                              │
└─────────────────────────────────────────────────────────────┘
┌─────────────────────────────────────────────────────────────┐
│                         Footer                               │
└─────────────────────────────────────────────────────────────┘
```

## 2.4.3 功能详情

### 错误提示

| 元素 | 内容 |
|------|------|
| 错误代码 | "404"（大号字体，醒目颜色） |
| 错误标题 | "页面不存在" |
| 错误描述 | "您访问的页面已失效或不存在" |

### 导航选项

| 按钮 | 链接 | 说明 |
|------|------|------|
| 返回首页 | `/` | 返回网站首页 |
| 浏览博客 | `/blog` | 跳转至博客列表 |
| 关于我 | `/about` | 跳转至关于页面 |

## 2.4.4 设计风格

- 简洁明了，不显示过多技术信息
- 使用品牌主色调
- 提供明确的返回路径
- 可选：添加一些趣味性的 404 插图/动画

## 2.4.5 SEO 元数据

| 标签 | 内容 |
|------|------|
| title | `404 - 页面未找到` |
| robots | noindex（404 页面不需要被索引） |

## 2.4.6 验收标准

- [ ] 404 页面在访问无效路径时正确显示
- [ ] 错误信息清晰易懂
- [ ] 导航按钮正常工作
- [ ] 页面设计简洁友好
- [ ] 响应式布局
