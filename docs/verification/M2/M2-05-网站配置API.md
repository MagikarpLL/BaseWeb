# M2-05 - 网站配置 API

> 对应任务: [01-迭代计划.md](../../plan/01-迭代计划.md#m2-05-网站配置-api)
> 涉及 Spec: [02-API契约.md](../../spec/02-API契约.md)

## 验证目标

管理员可读取和更新网站配置。

## 测试用例

### TC-01: 获取网站配置

**前置条件：**
- 管理员已登录

**测试步骤：**
1. GET `/api/admin/settings`
2. 验证返回配置数据

**预期结果：**
- 返回包含 site_name、site_description、social_links 等字段

### TC-02: 更新网站配置

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. PUT `/api/admin/settings` 请求体 `{"site_name": "新网站名称"}`
2. 验证返回更新后的数据

**预期结果：**
- site_name 已更新
- 更新时间已记录

### TC-03: 上传图片

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. POST `/api/admin/settings/upload` 上传图片文件
2. 验证返回图片 URL

**预期结果：**
- 返回图片 URL
- 图片已存储

## 验收标准

- [ ] TC-01: 获取网站配置通过
- [ ] TC-02: 更新网站配置通过
- [ ] TC-03: 上传图片通过
- [ ] 所有测试用例通过
