# 02-03-工具集页面

## 2.3.1 功能概述

工具集页面展示和提供各种实用在线小工具，供访客免费使用。

## 2.3.3 功能详情

### 工具卡片

| 字段 | 说明 |
|------|------|
| 工具名称 | 显示名称，最多 20 字符 |
| 工具图标 | Font Awesome 图标 |
| 工具描述 | 功能描述，最多 80 字符 |
| 分类标签 | 所属分类 |
| 链接 | 点击跳转至工具详情页 |

### 分类导航

| 分类 | 说明 |
|------|------|
| 全部 | 显示所有工具 |
| 格式化工具 | JSON、XML、HTML 格式化等 |
| 转换工具 | 时间戳、编码转换等 |
| 编码工具 | URL 编码、Base64 等 |
| 生成工具 | 密码生成、UUID 生成等 |
| 其他 | 不属于上述分类的工具 |

### 工具详情页（子页面）

每个工具在 `/tools/:toolSlug` 下有独立页面，包含：
- 工具标题和描述
- 工具输入区域
- 工具操作按钮
- 工具输出区域
- 使用说明

|----------|------|------|------|
| JSON 格式化 | json-formatter | 格式化工具 | JSON 字符串格式化、压缩、校验 |
| URL 编码 | url-encode | 编码工具 | URL 组件编码/解码 |
| Base64 编解码 | base64 | 编码工具 | Base64 字符串编码/解码 |
| 时间戳转换 | timestamp | 转换工具 | Unix 时间戳与日期互转 |
| 密码生成器 | password-generator | 生成工具 | 生成随机安全密码 |
| UUID 生成器 | uuid-generator | 生成工具 | 生成标准 UUID |
| 颜色转换 | color-converter | 转换工具 | HEX/RGB/HSL 颜色互转 |
| Cron 表达式 | cron-parser | 其他 | Cron 表达式解析和验证 |

## 2.3.6 用户交互

| 交互 | 行为 |
|------|------|
| 点击分类标签 | 筛选显示该分类下的工具 |
| 点击工具卡片 | 跳转至 `/tools/:slug` |
| 工具操作 | 实时处理，无页面刷新 |
