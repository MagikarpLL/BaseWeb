# 12-01-图片上传规格

> 对应 Requirement: [12-01-图片上传.md](../requirements/12-博客增强模块/12-01-图片上传.md)

## 12.1.1 概述 / 设计原则

|| 原则 | 说明 |
||------|------|
|| 前端压缩 | 图片压缩在前端完成，后端只负责存储 |
|| 无损压缩 | 先质量压缩，再尺寸压缩，保证画质 |
|| 目标大小 | 压缩至 1MB 以内，推荐 500KB-800KB |
|| 三种上传方式 | 工具栏、拖拽、粘贴截图 |

## 12.1.2 具体规格

### 12.1.2.1 后端接口（保持不变）

|| 接口 | 方法 | 说明 | 认证 |
||------|------|------|------|
|| `/api/admin/upload` | POST | 文件上传 | JWT (Admin) |

**请求格式**：
- Content-Type: `multipart/form-data`
- 字段名: `file`

**响应格式**：
```json
{
  "code": 200,
  "message": "File uploaded successfully",
  "data": {
    "url": "/uploads/1748601234567.jpg"
  }
}
```

**文件限制**：
- 允许格式: jpg, jpeg, png, gif, webp
- 最大大小: 5MB（前端已压缩，实际文件不会超过 1MB）

### 12.1.2.2 无损压缩算法

```typescript
/**
 * 图片压缩函数
 * @param file 原始文件
 * @param maxSizeKB 最大目标大小（KB），默认 1024（1MB）
 * @returns 压缩后的 Blob
 */
async function compressImage(file: File, maxSizeKB: number = 1024): Promise<Blob> {
  // 1. 创建图片元素
  const img = await loadImage(file)

  // 2. 获取图片尺寸
  let { width, height } = img

  // 3. 计算初始 quality
  let quality = 0.9
  const steps = [
    { width: 2000, quality: 0.9 },
    { width: 1800, quality: 0.85 },
    { width: 1600, quality: 0.8 },
    { width: 1400, quality: 0.75 },
    { width: 1200, quality: 0.7 },
    { width: 1000, quality: 0.65 },
    { width: 800, quality: 0.6 },
    { width: 600, quality: 0.55 },
    { width: 400, quality: 0.5 },
  ]

  // 4. 逐步压缩直到满足大小要求
  for (const step of steps) {
    if (width > step.width) {
      // 缩小尺寸
      const ratio = step.width / width
      width = step.width
      height = Math.round(height * ratio)
    }
    quality = step.quality

    // 5. 尝试压缩并检查大小
    const canvas = document.createElement('canvas')
    canvas.width = width
    canvas.height = height
    const ctx = canvas.getContext('2d')!
    ctx.drawImage(img, 0, 0, width, height)

    const blob = await canvasToBlob(canvas, file.type, quality)
    if (blob.size <= maxSizeKB * 1024) {
      return blob
    }
  }

  // 6. 最后尝试：最低质量 + 最小尺寸
  const canvas = document.createElement('canvas')
  canvas.width = 400
  canvas.height = Math.round(height * (400 / width))
  const ctx = canvas.getContext('2d')!
  ctx.drawImage(img, 0, 0, canvas.width, canvas.height)

  return canvasToBlob(canvas, file.type, 0.5)
}

function loadImage(file: File): Promise<HTMLImageElement> {
  return new Promise((resolve, reject) => {
    const img = new Image()
    img.onload = () => resolve(img)
    img.onerror = reject
    img.src = URL.createObjectURL(file)
  })
}

async function canvasToBlob(
  canvas: HTMLCanvasElement,
  type: string,
  quality: number
): Promise<Blob> {
  return new Promise((resolve, reject) => {
    canvas.toBlob(
      (blob) => {
        if (blob) resolve(blob)
        else reject(new Error('Canvas to Blob failed'))
      },
      type,
      quality
    )
  })
}
```

### 12.1.2.4 前端集成（新增）

在 `PostEdit.vue` 中添加以下功能：

```typescript
const toolbarConfig = {

```

#### 拖拽上传

```typescript
// 编辑器区域监听 dragover/drop 事件
function handleDragOver(e: DragEvent) {
  e.preventDefault()
  e.dataTransfer!.dropEffect = 'copy'
}

function handleDrop(e: DragEvent) {
  e.preventDefault()
  const files = e.dataTransfer?.files
  if (files?.length) {
    uploadImage(files[0])
  }
}
```

#### 粘贴截图

```typescript
// 编辑器区域监听 paste 事件
function handlePaste(e: ClipboardEvent) {
  const items = e.clipboardData?.items
  if (!items) return

  for (const item of items) {
    if (item.type.indexOf('image') !== -1) {
      const file = item.getAsFile()
      if (file) {
        uploadImage(file)
      }
    }
  }
}
```

#### 上传函数（含压缩）

```typescript
const MAX_FILE_SIZE = 1024 * 1024 // 1MB

async function uploadImage(file: File) {
  // 0. 验证文件类型
  const allowedTypes = ['image/jpeg', 'image/png', 'image/gif', 'image/webp']
  if (!allowedTypes.includes(file.type)) {
    ElMessage.error('不支持该文件格式')
    return
  }

  // 1. 如果文件超过 1MB，先压缩
  let fileToUpload = file
  if (file.size > MAX_FILE_SIZE) {
    try {
      fileToUpload = await compressImage(file, 800) // 目标 800KB
      console.log(`Compressed ${file.name} (${file.size} → ${fileToUpload.size})`)
    } catch (err) {
      ElMessage.error('图片压缩失败，将尝试原文件上传')
      fileToUpload = file
    }
  }

  // 2. 上传
  const formData = new FormData()
  formData.append('file', fileToUpload)

  try {
    const res = await settingsApi.uploadImage(formData)
    const url = res.data.data.url

    // 3. 插入 Markdown 图片语法到光标位置
    insertTextAtCursor(`![${file.name}](${url})`)
    ElMessage.success('图片上传成功')
  } catch (error) {
    ElMessage.error('图片上传失败')
  }
}
```
