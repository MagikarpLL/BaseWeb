# M2-10 - 前台工具集页面

> 对应任务: [01-迭代计划.md](../../plan/01-迭代计划.md#m2-10-前台工具集页面)
> 涉及 Spec: [09-前端路由设计.md](../../spec/09-前端路由设计.md)

## 验证目标

前台工具集页面支持分类筛选和卡片网格展示。

## 测试用例

### TC-01: 页面访问

**前置条件：**
- 无

**测试步骤：**
1. 访问 `/tools`
2. 验证页面加载

**预期结果：**
- 页面正常显示
- SEO meta 标签正确

### TC-02: 工具卡片显示

**前置条件：**
- 存在工具数据

**测试步骤：**
1. 访问 `/tools`
2. 验证工具卡片网格

**预期结果：**
- 卡片显示 name、description
- 卡片可点击

### TC-03: 分类筛选

**前置条件：**
- 存在多个分类的工具

**测试步骤：**
1. 访问 `/tools`
2. 点击分类筛选
3. 验证列表过滤

**预期结果：**
- 只显示该分类的工具
- 筛选条件可重置

## 验收标准

- [ ] TC-01: 页面访问通过
- [ ] TC-02: 工具卡片显示通过
- [ ] TC-03: 分类筛选通过
