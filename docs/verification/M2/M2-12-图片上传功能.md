# M2-12 - 图片上传功能

> 对应任务: [01-迭代计划.md](../../plan/01-迭代计划.md#m2-12-图片上传功能)
> 涉及 Spec: [02-API契约.md](../../spec/02-API契约.md)

## 验证目标

图片上传功能正常工作，返回图片 URL。

## 测试用例

### TC-01: 图片上传

**前置条件：**
- 管理员已登录

**测试步骤：**
1. POST `/api/admin/settings/upload` 上传 PNG 图片
2. 验证返回图片 URL

**预期结果：**
- 返回 200 和图片 URL
- 图片已存储

### TC-02: 图片格式验证

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 上传 JPG 图片
2. 验证返回

**预期结果：**
- 返回 200 和图片 URL

### TC-03: 图片大小限制

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 上传大于 5MB 的图片
2. 验证返回错误

**预期结果：**
- 返回 413 或类似错误码
- 提示图片过大
