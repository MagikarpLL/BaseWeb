# 02-04-404 页面

> 对应 Requirement: [02-04-404页面.md](../requirements/02-个人网站展示模块/02-04-404页面.md)

## 2.4.1 设计原则

| 原则 | 说明 |
|------|------|
| 友好提示 | 不显示技术细节，仅告知页面不存在 |
| 快速导航 | 提供明确返回路径，减少跳出 |
| 品牌一致 | 保持与整体网站风格一致 |

---

## 2.4.2 页面渲染规格

### 错误信息展示

| 元素 | 样式规格 |
|------|----------|
| 错误代码 | 字号 `120px`，字号 `bold`，颜色 `#667eea`，字体 `'Arial Black'` |
| 错误标题 | 字号 `2rem`，颜色 `#333`，居中 |
| 错误描述 | 字号 `1rem`，颜色 `#666`，居中 |

### 导航按钮

| 按钮 | 链接 | 样式规格 |
|------|------|----------|
| 返回首页 | `/` | 主按钮，背景 `#667eea`，白字 |
| 浏览博客 | `/blog` | 次按钮，边框 `#667eea`，蓝字 |
| 关于我 | `/about` | 次按钮，边框 `#667eea`，蓝字 |

**按钮尺寸**：宽度 `160px`，高度 `48px`，圆角 `8px`

**按钮间距**：垂直排列，间距 `16px`

### 布局规则

- 内容区垂直居中（`min-height: 60vh`）
- Header 仅显示 Logo，不显示导航菜单
- Footer 与前台其他页面一致

## 2.4.4 SEO 与 Meta

| 标签 | 值 |
|------|------|
| title | `404 - 页面未找到 - {网站名}` |
| robots | `noindex, nofollow` |

---

## 2.4.5 关键逻辑

### 404 触发场景

| 场景 | 处理方式 |
|------|----------|
| 前台路由无匹配 | Vue Router `/:pathMatch(.*)*` 捕获 |
| 文章 slug 无效 | 组件内调用 `router.replace('/404')` |
| 后台路由无匹配 | 同前台，由外层 `/:pathMatch` 捕获 |

---

## 2.4.6 验收标准

- [ ] 访问无效路径（如 `/xyz`）显示 404 页面
- [ ] 访问无效文章 slug（如 `/blog/non-existent`）重定向到 404
- [ ] 错误代码 "404" 醒目显示
- [ ] 错误标题和描述清晰易懂
- [ ] "返回首页" 按钮点击跳转 `/`
- [ ] "返回博客" 按钮点击跳转 `/blog`
- [ ] "关于我" 按钮点击跳转 `/about`
- [ ] 页面样式简洁友好
- [ ] 响应式布局：移动端字体适当缩小
- [ ] Meta 标签正确设置（title/robots）
