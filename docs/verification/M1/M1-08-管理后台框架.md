# M1-08 - 管理后台框架

> 对应任务: [01-迭代计划.md](../../plan/01-迭代计划.md#m1-08-管理后台框架侧边栏布局-子路由)
> 涉及 Spec: [09-前端路由设计.md](../../spec/09-前端路由设计.md)

## 验证目标

管理后台布局（侧边栏 + 主内容区）正常显示，子路由切换正常。

## 测试用例

### TC-01: 后台布局访问

**前置条件：**
- 管理员已登录

**测试步骤：**
1. 访问 `/admin`
2. 验证布局

**预期结果：**
- 左侧边栏显示
- 主内容区显示
- 顶部可能有面包屑或标题

### TC-02: 侧边栏菜单

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 检查侧边栏菜单项
2. 验证菜单项

**预期结果：**
- 显示仪表盘、文章管理、分类管理、标签管理、评论管理、配置管理等菜单
- 图标正常显示

### TC-03: 菜单项点击

**前置条件：**
- TC-02 已通过

**测试步骤：**
1. 点击"文章管理"菜单
2. 验证内容区切换

**预期结果：**
- 右侧内容区显示文章管理页面
- 侧边栏菜单项高亮

### TC-04: 子路由切换

**前置条件：**
- TC-03 已通过

**测试步骤：**
1. 在后台内点击不同菜单
2. 验证路由切换

**预期结果：**
- URL 变化
- 内容区正确切换

### TC-05: 响应式布局

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 缩小浏览器窗口到移动端宽度
2. 验证布局调整

**预期结果：**
- 侧边栏可能折叠为汉堡菜单
- 内容区自适应

### TC-06: 登出功能

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 点击登出或用户菜单
2. 验证登出

**预期结果：**
- 跳转到登录页
- 清除登录状态

- [ ] TC-01: 后台布局访问通过
- [ ] TC-02: 侧边栏菜单通过
- [ ] TC-03: 菜单项点击通过
- [ ] TC-04: 子路由切换通过

- [ ] TC-06: 登出功能通过
