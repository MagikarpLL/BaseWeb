# M1-13 - 前台首页

> 对应任务: [01-迭代计划.md](../../plan/01-迭代计划.md#m1-13-前台首页)
> 涉及 Spec: [09-前端路由设计.md](../../spec/09-前端路由设计.md)

## 验证目标

前台首页显示个人信息、最新文章、精选工具占位。

## 测试用例

### TC-01: 首页访问

**前置条件：**
- 前端服务已启动

**测试步骤：**
1. 访问 `/`
2. 验证页面加载

**预期结果：**
- 页面正常显示
- SEO meta 标签正确

### TC-02: Hero 区域

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 查看首页 Hero 区域
2. 验证显示

**预期结果：**
- 显示个人头像、名称、简介

### TC-03: 最新文章列表

**前置条件：**
- 存在已发布的文章

**测试步骤：**
1. 查看首页文章列表
2. 验证显示

**预期结果：**
- 显示最新 3 篇文章
- 每篇显示标题、摘要、发布日期

### TC-04: 最新文章点击

**前置条件：**
- TC-03 已通过

**测试步骤：**
1. 点击某篇文章的标题
2. 验证跳转

**预期结果：**
- 跳转到文章详情页

### TC-05: 精选工具区域

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 查看首页精选工具区域
2. 验证显示（可以是占位符）

**预期结果：**
- 区域存在
- 显示"敬请期待"或类似占位内容

## 验收标准

- [ ] TC-01: 首页访问通过
- [ ] TC-02: Hero 区域通过
- [ ] TC-03: 最新文章列表通过
- [ ] TC-04: 最新文章点击通过
- [ ] TC-05: 精选工具区域通过
- [ ] 所有测试用例通过
