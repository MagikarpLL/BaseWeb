## 10.1.1 功能概述

管理后台是管理员专用的内容管理系统，提供博客文章管理、分类标签管理、网站配置、评论审核、数据统计等功能。整个管理后台采用统一的侧边栏布局。

│  顶部栏: Logo  ·  网站名称  ·            管理员  [退出]     │

│          │                                                   │
│  侧边栏  │              主内容区                              │
│          │                                                   │
│  仪表盘   │                                                   │

│  文章管理 │                                                   │
│   · 列表  │                                                   │

│  标签管理 │                                                   │

### 侧边栏导航项

| 编号 | 导航项 | 路由 | 图标 | 说明 |
|------|--------|------|------|------|
| 1 | 仪表盘 | `/admin` | dashboard | 后台首页，显示概览统计 |
| 2 | 文章管理 | `/admin/posts` | document | 文章列表 |
| 3 | 新建文章 | `/admin/posts/new` | plus | 创建新文章 |
| 4 | 分类管理 | `/admin/categories` | folder | 分类 CRUD |
| 5 | 标签管理 | `/admin/tags` | tag | 标签 CRUD |
| 6 | 评论管理 | `/admin/comments` | chat | 评论审核管理 |
| 7 | 网站配置 | `/admin/settings` | settings | 网站配置 |
| 8 | 数据统计 | `/admin/analytics` | chart | 访问统计面板 |

### 侧边栏行为

- 当前页面导航项高亮
- 支持折叠/展开
- 移动端自动收起，点击汉堡菜单展开
- 退出按钮在顶部栏右侧

## 10.1.4 全局交互规则

### 表单操作

| 操作 | 行为 |
|------|------|
| 保存 | 成功后显示 Toast 提示，停留在当前页 |
| 取消 | 弹出确认"是否放弃编辑？"，确认后返回列表页 |
| 删除 | 弹出确认"确认删除？此操作不可恢复"，确认后删除并刷新列表 |

### 列表页通用规则

| 规则 | 说明 |
|------|------|
| 默认分页 | 每页 10 条 |
| 空状态 | 显示"暂无数据"插图 + 引导文案 |
| 加载状态 | 显示骨架屏或 loading 动画 |
| 删除后 | 自动刷新列表，若当前页无数据则回到上一页 |

### 响应式适配

| 断点 | 行为 |
|------|------|
| >= 1024px (桌面) | 侧边栏常驻展开，主内容区自适应宽度 |
| 768-1023px (平板) | 侧边栏折叠为图标模式，鼠标悬停展开 |
| < 768px (手机) | 侧边栏隐藏，顶部显示汉堡菜单，点击从左滑出 |

## 10.1.5 权限与安全

| 规则 | 说明 |
|------|------|
| 登录验证 | 所有 `/admin/*` 页面需验证 JWT Token |
| 未登录处理 | 重定向到 `/login`，并记录来源页面，登录后跳回 |
| Token 过期 | 自动尝试 Refresh，失败则跳转登录页 |
| 操作审计 | 管理操作（增删改）记录操作日志（后续迭代） |
