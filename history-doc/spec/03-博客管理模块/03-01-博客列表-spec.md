# 03-01-博客列表

> 对应 Requirement: [03-01-博客列表.md](../requirements/03-博客管理模块/03-01-博客列表.md)

## 3.1.1 设计原则

| 原则 | 说明 |
|------|------|
| 高效浏览 | 支持分类/标签筛选和排序，快速定位文章 |
| 分页加载 | 每页 10 篇，避免长列表性能问题 |
| SEO 友好 | 列表页需完整的 meta 标签和结构化数据 |

---

## 3.1.2 数据模型

### 文章列表 API

**接口**: `GET /api/blog/posts`

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| page | int | 1 | 页码 |
| size | int | 10 | 每页数量 |
| category | string | - | 分类 slug 筛选 |
| tag | string | - | 标签 slug 筛选 |
| sort | string | latest | 排序：latest/updated/popular |

**响应**:

```json
{
  "code": 200,
  "data": {
    "posts": [
      {
        "id": 1,
        "title": "Vue 3 Composition API 最佳实践",
        "slug": "vue3-composition-api-best-practices",
        "excerpt": "深入解析 Vue 3 Composition API 的使用方法...",
        "publishedAt": "2026-04-20",
        "readingTime": 5,
        "category": { "id": 1, "name": "前端", "slug": "frontend" },
        "tags": [{ "id": 1, "name": "Vue.js", "slug": "vuejs" }],

      }
    ],
```

## 3.1.3 页面渲染规格

### 筛选区域

| 元素 | 渲染规则 |
|------|----------|
| 分类下拉 | 显示 `categories` 列表 + "全部分类" 选项 |
| 标签下拉 | 显示 `tags` 列表 + "全部标签" 选项 |
| 排序下拉 | 选项：最新发布 / 最新更新 / 阅读量最高 |
| 默认选中 | category=all, tag=all, sort=latest |

**交互**：
- 切换筛选 → URL 参数更新 → 列表刷新（无页面跳转）
- 筛选状态显示在 URL：`/blog?category=frontend&tag=vuejs&sort=latest`

### 文章卡片

| 字段 | 渲染规则 |
|------|----------|
| 标题 | 最多 `100` 字符，超出截断 |
| 摘要 | 最多 `200` 字符，超出截断 |
| 发布日期 | 格式 `YYYY-MM-DD` |
| 阅读时间 | 显示 `{n} 分钟阅读` |
| 分类 | 显示分类名称，可点击跳转筛选 |
| 标签 | 显示最多 `3` 个标签，超出显示 `+N` |
| 阅读量 | 数字格式化，如 `1,234` |

**卡片样式**：
- 边框：`1px solid #e0e0e0`，圆角 `8px`
- Hover：阴影加深，上移 `2px`
- 封面图：可选，高度 `160px`，`object-fit: cover`

### 分页导航

| 元素 | 渲染规则 |
|------|----------|
| 页码显示 | 当前页前后各 2 页 + 首页末页 |
| 当前页 | 背景 `#667eea`，白字 |
| 省略符 | `...` 显示在连续页码之间 |
| 总页数 | 显示 `共 N 页` |

**示例**: `< 1 2 3 ... 10 >`

## 3.1.4 关键逻辑

### 筛选状态同步 URL

```typescript
// 筛选变化 → 更新 URL
router.push({ query: { category, tag, sort, page: 1 } })

// 页面加载 → 从 URL 恢复筛选状态
const { category, tag, sort, page } = route.query
```

### 分页边界处理

| 场景 | 处理 |
|------|------|
| 请求页 > 总页数 | 自动修正为总页数 |
| 删除最后一页文章 | 自动跳转到上一页 |
