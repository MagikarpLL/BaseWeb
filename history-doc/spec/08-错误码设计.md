# 08-错误码设计

> 本文档定义后端 API 的业务错误码体系和前端错误处理策略。
> 对应 Requirement: [05-01-错误处理.md](../requirements/05-错误处理模块/05-01-错误处理.md)

## 8.1 错误响应格式

所有 API 错误响应遵循统一格式：

```json
{
  "code": 40001,
  "message": "文章标题不能为空",
  "data": null,
  "timestamp": 1745404800000
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| code | int | 业务错误码，前 3 位为 HTTP 状态码，后 2 位为业务细分码 |
| message | string | 面向开发者的错误描述（前端可直接展示给用户） |
| data | null | 错误响应始终为 null |
| timestamp | long | 服务器时间戳 |

## 8.2 错误码编码规则

```
  4 0 0 0 1
  │ │ │ │ └── 业务细分码 (01-99)
  │ │ │ └──── 固定 0
  │ │ └────── 固定 0
  │ └───────── HTTP 状态码第二位
  └─────────── HTTP 状态码第一位（4=客户端错误，5=服务端错误）
```

**简写规则**：`{HTTP状态码}{业务细分码}`，如 `40001` = HTTP 400 + 业务码 01。

## 8.3 错误码清单

### 通用错误（400xx）

| 错误码 | HTTP | 说明 | 前端处理 |
|--------|------|------|----------|
| 40001 | 400 | 请求参数校验失败 | 显示 message 中的具体校验信息 |
| 40002 | 400 | 请求体格式错误（JSON 解析失败） | Toast "请求数据格式错误" |
| 40003 | 400 | 分页参数越界 | Toast "页码超出范围" |

### 认证错误（401xx）

| 错误码 | HTTP | 说明 | 前端处理 |
|--------|------|------|----------|
| 40101 | 401 | 未提供 Token | 跳转登录页 |
| 40102 | 401 | Token 无效或已过期 | 尝试刷新 Token |
| 40103 | 401 | Refresh Token 无效或已过期 | 跳转登录页 |
| 40104 | 401 | 账户已被锁定 | 表单显示"账户已锁定，请 15 分钟后重试" |

### 权限错误（403xx）

| 错误码 | HTTP | 说明 | 前端处理 |
|--------|------|------|----------|
| 40301 | 403 | 无权访问该资源 | Toast "无操作权限" |
| 40302 | 403 | 非管理员访问管理接口 | 跳转登录页 |

### 资源不存在（404xx）

| 错误码 | HTTP | 说明 | 前端处理 |
|--------|------|------|----------|
| 40401 | 404 | 文章不存在 | 重定向到 404 页面 |
| 40402 | 404 | 分类不存在 | Toast "分类不存在" |
| 40403 | 404 | 标签不存在 | Toast "标签不存在" |
| 40404 | 404 | 配置项不存在 | Toast "配置项不存在" |
| 40405 | 404 | 评论不存在 | Toast "评论不存在" |

### 业务冲突（409xx）

| 错误码 | HTTP | 说明 | 前端处理 |
|--------|------|------|----------|
| 40901 | 409 | 文章 slug 已存在 | 高亮 slug 字段，提示"该链接地址已被使用" |
| 40902 | 409 | 分类 slug 已存在 | 高亮 slug 字段，提示"该分类标识已存在" |
| 40903 | 409 | 标签 slug 已存在 | 高亮 slug 字段，提示"该标签标识已存在" |
| 40904 | 409 | 分类下仍有文章，无法删除 | Toast "该分类下仍有文章，请先转移文章" |

### 文件上传错误（413xx）

| 错误码 | HTTP | 说明 | 前端处理 |
|--------|------|------|----------|
| 41301 | 413 | 文件大小超过限制 | Toast "文件大小超过限制（最大 5MB）" |
| 41302 | 413 | 不支持的文件类型 | Toast "不支持的文件格式（仅 JPG/PNG/GIF/WebP）" |

### 评论相关错误（422xx）

| 错误码 | HTTP | 说明 | 前端处理 |
|--------|------|------|----------|
| 42201 | 422 | 验证码错误 | 高亮验证码字段，提示"验证码错误" |
| 42202 | 422 | 验证码已过期 | 提示"验证码已过期，请刷新" |
| 42203 | 422 | 评论内容含禁止内容 | Toast "评论内容包含不允许的内容" |

### 服务端错误（500xx）

| 错误码 | HTTP | 说明 | 前端处理 |
|--------|------|------|----------|
| 50001 | 500 | 服务器内部错误 | Toast "服务器异常，请稍后重试" |
| 50002 | 500 | 数据库操作失败 | Toast "操作失败，请稍后重试" |
| 50003 | 500 | 文件写入失败 | Toast "文件保存失败，请重试" |

## 8.4 前端错误处理策略

### 请求拦截器处理流程

```
API 响应
  │
  ├── 网络错误（无响应）
  │   └── Toast "网络异常，请检查网络连接"
  │
  ├── HTTP 200 + code === 200
  │   └── 正常返回 data
  │
  ├── HTTP 401
  │   ├── 40102 (Token 过期) → 尝试 Refresh
  │   │   ├── 刷新成功 → 重试原请求
  │   │   └── 刷新失败 → 跳转 /login
  │   ├── 40103 (Refresh 过期) → 跳转 /login
  │   └── 40104 (账户锁定) → 登录页显示锁定提示
  │
  ├── HTTP 403
  │   └── Toast "无操作权限"
  │
  ├── HTTP 404
  │   ├── 40401 (文章不存在) → 路由跳转 404 页面
  │   └── 其他 → Toast 显示 message
  │
  ├── HTTP 409
  │   └── 表单级错误：高亮对应字段
  │
  ├── HTTP 413/422
  │   └── Toast 显示 message
  │
  └── HTTP 500
      └── Toast "服务器异常，请稍后重试"
```

### Axios 拦截器伪代码

```typescript
// 响应拦截器
axios.interceptors.response.use(
  (response) => {
    const { code, data, message } = response.data;
    if (code === 200) return data;
    return Promise.reject({ code, message });
  },
  async (error) => {
    if (!error.response) {
      // 网络错误
      showToast('网络异常，请检查网络连接');
      return Promise.reject(error);
    }

    const { status, data } = error.response;
    const { code, message } = data;

    if (status === 401) {
      if (code === 40102) {
        // Token 过期，尝试刷新
        try {
          await refreshToken();
          return retryOriginalRequest(error.config);
        } catch {
          router.push('/login');
          showToast('登录已过期，请重新登录');
        }
      }
      router.push('/login');
    } else if (status === 409) {
      // 表单冲突错误，由调用方处理
      return Promise.reject({ code, message, isFormError: true });
    } else if (status === 404 && code === 40401) {
      router.push('/404');
    } else {
      showToast(message || '操作失败');
    }

    return Promise.reject({ code, message });
  }
);
```

## 8.5 后端错误码枚举

```java
public enum ErrorCode {
    // 通用
    BAD_REQUEST(40001, "请求参数校验失败"),
    INVALID_JSON(40002, "请求数据格式错误"),
    PAGE_OUT_OF_RANGE(40003, "页码超出范围"),

    // 认证
    UNAUTHORIZED(40101, "未认证，请先登录"),
    TOKEN_EXPIRED(40102, "Token 已过期"),
    REFRESH_TOKEN_EXPIRED(40103, "登录已过期，请重新登录"),
    ACCOUNT_LOCKED(40104, "账户已锁定，请稍后重试"),

    // 权限
    FORBIDDEN(40301, "无操作权限"),
    ADMIN_REQUIRED(40302, "需要管理员权限"),

    // 资源不存在
    POST_NOT_FOUND(40401, "文章不存在"),
    CATEGORY_NOT_FOUND(40402, "分类不存在"),
    TAG_NOT_FOUND(40403, "标签不存在"),
    SETTING_NOT_FOUND(40404, "配置项不存在"),
    COMMENT_NOT_FOUND(40405, "评论不存在"),

    // 冲突
    DUPLICATE_POST_SLUG(40901, "文章链接地址已存在"),
    DUPLICATE_CATEGORY_SLUG(40902, "分类标识已存在"),
    DUPLICATE_TAG_SLUG(40903, "标签标识已存在"),
    CATEGORY_HAS_POSTS(40904, "该分类下仍有文章"),

    // 文件上传
    FILE_TOO_LARGE(41301, "文件大小超过限制"),
    FILE_TYPE_NOT_ALLOWED(41302, "不支持的文件类型"),

    // 评论
    CAPTCHA_INVALID(42201, "验证码错误"),
    CAPTCHA_EXPIRED(42202, "验证码已过期"),
    COMMENT_BLOCKED(42203, "评论内容包含不允许的内容"),

    // 服务端
    INTERNAL_ERROR(50001, "服务器内部错误"),
    DATABASE_ERROR(50002, "数据库操作失败"),
    FILE_WRITE_ERROR(50003, "文件保存失败");

    private final int code;
    private final String message;

    ErrorCode(int code, String message) {
        this.code = code;
        this.message = message;
    }

    public int getCode() { return code; }
    public String getMessage() { return message; }
}
```

## 8.6 验证码错误码 vs HTTP 状态码映射

| 错误码范围 | HTTP 状态码 | 说明 |
|------------|------------|------|
| 40001-40099 | 400 | 请求参数/格式错误 |
| 40101-40199 | 401 | 认证相关错误 |
| 40301-40399 | 403 | 权限相关错误 |
| 40401-40499 | 404 | 资源不存在 |
| 40901-40999 | 409 | 业务冲突 |
| 41301-41399 | 413 | 文件上传错误 |
| 42201-42299 | 422 | 业务逻辑校验失败 |
| 50001-50099 | 500 | 服务端内部错误 |
