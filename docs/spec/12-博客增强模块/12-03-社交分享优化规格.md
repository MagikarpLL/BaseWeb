## 12.3.2 具体规格

### 12.3.2.1 前端元数据完善

在 `BlogDetail.vue` 的 `useHead()` 中补充以下 meta 标签：

**当前已有**：
- og:title ✅
- og:description ✅
- og:image ✅
- og:type ✅
- article:published_time ✅
- article:modified_time ✅
- article:author ✅
- twitter:card ✅
- twitter:title ✅
- twitter:description ✅
- twitter:image ✅

**需要补充**：

```typescript
// BlogDetail.vue useHead() meta 配置
{name: 'og:url', content: currentUrl.value},
{property: 'article:section', content: post.value.categoryName},
{property: 'article:tag', content: post.value.tags?.map(t => t.name).join(',')},
{name: 'twitter:site', content: '@yourTwitterHandle'},
```

### 12.3.2.2 JSON-LD 结构化数据

当前已有的 Article Schema：
```json
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "文章标题",
  "description": "文章摘要",
  "image": "封面图 URL",
  "datePublished": "发布时间",
  "dateModified": "修改时间",
  "author": {
    "@type": "Person",
    "name": "作者名"
  }
}
```

**需要补充**：

```typescript
// 在 useBlogPostSchema 函数中补充
mainEntityOfPage: {
  "@type": "WebPage",
  "@id": currentUrl.value  // og:url 的完整 URL
}
```

### 12.3.2.3 og:image 尺寸要求

|| 平台 | 推荐尺寸 | 最小尺寸 |
||------|----------|----------|
| Facebook | 1200x630 | 600x315 |
| Twitter | 1200x675 | 300x157 |

建议在图片上传或设置封面图时进行尺寸校验。

### 12.3.2.4 配置文件

在 `SiteSettings` 中添加可选配置：

```typescript
interface SiteSettings {
  // ... 现有字段
  social: {
    twitter?: string  // Twitter @用户名
  }
}
```
