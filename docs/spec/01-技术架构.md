# 01-技术架构

## 1.1 技术栈选型

### 后端技术栈

| 组件 | 选择 | 版本 | 说明 |
|------|------|------|------|
| **JDK** | OpenJDK | 17 | LTS 版本，稳定可靠 |
| **框架** | Spring Boot | 3.x | 当前主流，企业级标准 |
| **ORM** | MyBatis | 3.x | 轻量级 SQL 控制，可编写复杂查询 |
| **数据库** | PostgreSQL | 15+ | JSON 支持好，适合内容型数据 |
| **安全** | Spring Security + JWT | - | 无状态认证，适合 SPA |

### 前端技术栈

| 组件 | 选择 | 版本 | 说明 |
|------|------|------|------|
| **框架** | Vue 3 | 3.5+ | Composition API，Vapor Mode |
| **构建** | Vite | 6.x | 极速 HMR，生产构建快 |
| **路由** | Vue Router | 4.x | History 模式 |
| **状态** | Pinia | 2.x | 官方推荐，轻量状态管理 |
| **UI 库** | Element Plus | 2.x | 稳定，组件丰富 |
| **HTTP** | Axios | 1.x | 拦截器支持 JWT 刷新 |
| **SEO** | @unhead/vue | 2026 | Meta 标签管理 |
| **预渲染** | vite-ssg | - | 静态预渲染 |
| **Markdown** | @milkdown/vue | - | 富文本编辑 |
| **图表** | ECharts | - | 数据可视化 |

## 1.3 项目目录结构

### 后端结构

```
backend/
├── pom.xml
└── src/main/
    ├── java/com/example/personalwebsite/
    │   ├── config/              # 配置类
    │   ├── controller/          # REST 控制器
    │   │   ├── public/          # 公开接口
    │   │   ├── admin/           # 管理接口
    │   │   ├── auth/            # 认证接口
    │   │   └── analytics/       # 统计接口
    │   ├── service/             # 业务逻辑
    │   ├── mapper/              # MyBatis Mapper
    │   ├── model/               # 实体 & DTO
    │   ├── exception/           # 异常处理
    │   └── security/            # 安全相关
    └── resources/
        ├── mapper/              # MyBatis XML
        ├── application.yml
        └── public/              # Vue 构建产物
```

### 前端结构

```
frontend/
├── package.json
├── vite.config.ts
└── src/
    ├── api/                     # API 调用
    ├── assets/                  # 静态资源
    ├── components/              # 组件
    │   ├── common/              # 基础组件
    │   ├── blog/                # 博客组件
    │   └── tools/               # 工具组件
    ├── composables/             # 组合式逻辑
    ├── layouts/                 # 布局
    ├── router/                  # 路由
    ├── stores/                  # Pinia 状态
    ├── utils/                   # 工具函数
    └── views/                   # 页面
        ├── public/              # 公开页面
        └── admin/               # 管理后台
```
