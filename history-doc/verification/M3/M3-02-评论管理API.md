# M3-02 - 评论管理 API（审核/回复/删除）

> 对应任务: [01-迭代计划.md](../../plan/01-迭代计划.md#m3-02-评论管理-api审核回复删除)
> 涉及 Spec: [02-API契约.md](../../spec/02-API契约.md)

## 验证目标

管理员可审核、回复、删除评论。

## 测试用例

### TC-01: 获取待审核评论

**前置条件：**
- 管理员已登录
- 存在待审核评论

**测试步骤：**
1. GET `/api/admin/comments?status=pending`
2. 验证返回待审核评论

**预期结果：**
- 返回待审核评论列表
- 每条包含 content、author、post_id

### TC-02: 审核评论

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. PUT `/api/admin/comments/{id}/status` 请求体 `{"status": "approved"}`
2. 验证返回更新后的状态

**预期结果：**
- 状态已更新为 approved

### TC-03: 回复评论

**前置条件：**
- TC-02 已通过

**测试步骤：**
1. POST `/api/admin/comments/{id}/reply` 请求体 `{"content": "管理员回复"}`
2. 验证返回回复数据

**预期结果：**
- 回复创建成功
- parent_id 指向原评论

### TC-04: 删除评论

**前置条件：**
- TC-02 已通过

**测试步骤：**
1. DELETE `/api/admin/comments/{id}`
2. 验证返回 204

**预期结果：**
- 删除成功
- 数据库中已软删除
