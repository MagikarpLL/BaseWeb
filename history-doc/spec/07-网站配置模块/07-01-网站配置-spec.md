## 7.1.1 设计原则

| 原则 | 说明 |
|------|------|
| 集中管理 | 所有网站配置在单一后台界面完成 |
| 实时生效 | 配置修改后网站各页面立即更新 |
| 数据驱动 | 配置存储在数据库，不硬编码 |

**接口**: `GET /api/admin/settings`

```json
{
  "code": 200,
  "data": {
    "siteName": "张三的技术站",
    "siteUrl": "https://yoursite.com",
    "siteDescription": "全栈工程师的技术博客",
    "name": "张三",
    "title": "全栈工程师",
    "avatar": "/uploads/avatar.jpg",
    "bio": "热爱技术，专注于 Web 开发",
    "location": "上海",
    "socialLinks": [
      { "platform": "github", "url": "https://github.com/zhangsan", "isShow": true }
    ],
    "aboutContent": "## 个人介绍\n\n这是我的详细介绍...",
    "skills": [
      { "name": "Vue.js", "level": 90, "category": "frontend" }
    ],
    "experiences": {
      "education": [],
      "work": []
    },
    "projects": []
  }

```

### 更新配置

**接口**: `PUT /api/admin/settings`

```json
{
  "siteName": "张三的技术站",
  "name": "张三",
  "avatar": "/uploads/avatar.jpg",
  "socialLinks": [...],
  "aboutContent": "...",
  "skills": [...],
  "experiences": {...},
  "projects": [...]
}
```

### 图片上传

**接口**: `POST /api/admin/settings/upload`

**请求**: `multipart/form-data`

**响应**:
```json
{
  "code": 200,
  "data": {
    "url": "/uploads/avatar-1745404800.jpg",
    "filename": "avatar-1745404800.jpg"
  }
}
```

## 7.1.3 页面渲染规格

### Tab 导航

| Tab | 内容 |
|-----|------|
| 基础信息 | siteName / siteUrl / siteDescription |
| 个人信息 | name / title / avatar / bio / location |
| 社交链接 | 平台列表，可增删改 |
| 关于页 | aboutContent / skills / experiences / projects |
| SEO 设置 | siteKeywords / ogImage |

### 表单字段

**基础信息**：
| 字段 | 类型 | 规则 |
|------|------|------|
| siteName | 文本 | 必填，最多 50 字符 |
| siteUrl | URL | 必填，有效 URL 格式 |
| siteDescription | 文本 | 最多 200 字符 |

**个人信息**：
| 字段 | 类型 | 规则 |
|------|------|------|
| name | 文本 | 必填，最多 30 字符 |
| title | 文本 | 必填，最多 50 字符 |
| avatar | 图片上传 | 支持 JPG/PNG/GIF/WebP，最大 5MB |
| bio | 文本 | 最多 200 字符 |
| location | 文本 | 最多 50 字符 |

**社交链接**：
| 字段 | 类型 | 规则 |
|------|------|------|
| platform | 下拉 | github/twitter/email/linkedin/bilibili |
| url | URL | 必填，有效 URL 格式 |
| isShow | 开关 | 是否显示 |

## 7.1.4 关键逻辑

### 技能编辑

```typescript
// 技能数据结构
interface Skill {
  name: string       // 技能名称
  level: number      // 熟练度 0-100
  category: string    // 分类：frontend/backend/database/tools
}

// 添加技能
function addSkill(skill: Skill) {
  skills.push(skill)
}

// 删除技能
function removeSkill(index: number) {
  skills.splice(index, 1)
}

// 编辑技能
function updateSkill(index: number, skill: Skill) {
  skills[index] = skill
}
```

### 项目编辑

```typescript
// 项目数据结构
interface Project {
  name: string
  description: string
  techStack: string[]
  url?: string
  image?: string
  featured: boolean
}

// 添加项目
function addProject(project: Project) {
  projects.push(project)
}

// 删除项目
function removeProject(index: number) {
  projects.splice(index, 1)
}
```

### 图片上传处理

```typescript
async function uploadImage(file: File, type: 'avatar' | 'cover' | 'project') {
  const formData = new FormData()
  formData.append('file', file)
  formData.append('type', type)

  const res = await api.post('/api/admin/settings/upload', formData, {
    headers: { 'Content-Type': 'multipart/form-data' }
  })

  return res.data.url
}
```

## 7.1.6 验收标准

- [ ] 管理员可以编辑所有配置项（基础信息/个人信息/社交链接/关于页）
- [ ] 配置修改后保存成功
- [ ] 保存后网站各页面立即反映最新配置
- [ ] 图片上传功能正常（支持拖拽上传）
- [ ] 社交链接支持多平台（github/twitter/email/linkedin/bilibili）
- [ ] 技能列表支持增删改
- [ ] 经历列表支持增删改
- [ ] 项目列表支持增删改
- [ ] 所有配置数据存储在数据库 site_settings 表
- [ ] 非管理员用户无法访问配置页面
