# M3-06 - 前台埋点 SDK

> 对应任务: [01-迭代计划.md](../../plan/01-迭代计划.md#m3-06-前台埋点-sdk页面访问uv-pv上报)
> 涉及 Spec: [09-前端路由设计.md](../../spec/09-前端路由设计.md)

## 验证目标

前台页面自动上报埋点数据（页面访问、UV、PV）。

## 测试用例

### TC-01: 页面访问上报

**前置条件：**
- 已集成埋点 SDK

**测试步骤：**
1. 访问前台页面
2. 检查网络请求

**预期结果：**
- 自动发送 `/api/analytics/track` 请求
- 请求包含 page、visitor_id、event

### TC-02: UV 识别

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 首次访问页面
2. 清除 localStorage（或使用新浏览器）
3. 再次访问

**预期结果：**
- 两次访问生成不同的 visitor_id
- 相同 visitor_id 的访问只计一次 UV

### TC-03: PV 统计

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 访问页面 A
2. 跳转到页面 B
3. 验证两次访问均记录

**预期结果：**
- PV = 2
- UV = 1（相同 visitor_id）
