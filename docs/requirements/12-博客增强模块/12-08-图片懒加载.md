# 12-08-图片懒加载

## 12.8.1 功能概述

对博客文章中的图片启用懒加载，当图片进入视口时才加载，提升页面加载性能和用户体验。

## 12.8.2 页面布局

```
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│                      文章内容                                 │
│                                                              │
│   图片1（已加载）                                            │
│                                                              │
│   ┌─────────────────────────────────────────────────────┐   │
│   │                    占位图                             │   │ ← 视口外图片
│   │                    (loading...)                      │   │
│   └─────────────────────────────────────────────────────┘   │
│                                                              │
│   图片2（已加载）                                            │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

## 12.8.3 功能详情

### 12.8.3.1 懒加载实现

使用 Intersection Observer API 监听图片进入视口：

|| 阶段 | 处理方式 |
||------|----------|
| 初始状态 | `<img data-src="真实URL" class="lazy" src="占位图">` |
| 进入视口 | 将 `data-src` 赋值给 `src` 开始加载 |
| 加载完成 | 添加 `loaded` 类触发淡入动画 |

### 12.8.3.2 占位图

|| 项目 | 说明 |
||------|------|
|| 占位图类型 | 灰色背景 + loading 图标 |
|| 尺寸 | 与原图一致，防止页面抖动 |
|| 加载完成 | 淡入切换到真实图片 |

### 12.8.3.3 样式定义

> 样式定义和 CSS 类名见 Spec: [12-08-图片懒加载规格.md](../../spec/12-博客增强模块/12-08-图片懒加载规格.md)

### 12.8.3.4 降级处理

|| 浏览器 | 处理方式 |
||--------|----------|
|| 支持 Intersection Observer | 懒加载实现 |
|| 不支持（IE11） | 正常加载所有图片 |

### 12.8.3.5 影响范围

|| 位置 | 是否懒加载 |
||------|-----------|
|| 文章内容图片 | ✅ 懒加载 |
|| 封面图 | ❌ 首屏图片，正常加载 |
|| 相关文章缩略图 | ✅ 懒加载 |
|| 作者头像 | ❌ 小图片，正常加载 |

## 12.8.4 验收标准

- [ ] 文章内图片实现懒加载
- [ ] 滚动到图片位置时才加载
- [ ] 加载时有占位图显示
- [ ] 图片加载完成后有淡入效果
- [ ] 不影响现有图片显示
- [ ] 浏览器不支持时正常降级
- [ ] 页面性能有明显提升（LCP）
