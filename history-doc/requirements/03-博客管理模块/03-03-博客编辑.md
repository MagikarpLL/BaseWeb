## 3.3.3 功能详情

### 文章编辑表单

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 标题 | 文本输入 | ✅ | 最多 200 字符 |
| 分类 | 下拉选择 | ✅ | 必须选择 |
| 标签 | 多选 | ❌ | 最多选择 5 个 |
| 摘要 | 文本域 | ❌ | 最多 500 字符，不填则自动提取 |
| 内容 | Markdown 编辑器 | ✅ | Markdown 格式 |
| 状态 | 单选 | ✅ | 草稿 / 已发布 |
| 封面图 | 图片上传 | ❌ | 建议尺寸 1200x630px |

### Markdown 编辑器功能

| 功能 | 说明 |
|------|------|
| 实时预览 | 左编辑右预览分屏显示 |
| 工具栏 | 粗体、斜体、标题、链接、图片、代码块等 |
| 语法高亮 | 编辑区代码语法高亮 |
| 自动保存 | 每 30 秒自动保存到本地 |

### 操作按钮

| 按钮 | 说明 |
|------|------|
| 发布 | 保存文章并设置为"已发布"状态 |
| 保存草稿 | 保存文章但保持"草稿"状态 |
| 预览 | 新窗口打开预览页面 |

### 文章列表功能

| 功能 | 说明 |
|------|------|
| 列表展示 | 标题、状态、分类、发布时间、操作 |
| 搜索 | 按标题关键词搜索 |
| 筛选 | 按分类、按状态筛选 |
| 排序 | 按发布时间、阅读量排序 |
| 编辑 | 点击跳转编辑页面 |
| 删除 | 确认后删除（软删除） |
| 置顶 | 设置/取消置顶 |

## 3.3.4 权限控制

| 角色 | 权限 |
|------|------|
| 访客 | ❌ 无访问权限 |
| 管理员 | ✅ 全部权限 |

未登录用户访问 `/admin` 路径时，重定向至登录页。

## 3.3.5 验收标准

- [ ] 管理员可以创建新文章
- [ ] 管理员可以编辑已有文章

- [ ] Markdown 编辑器工具栏功能正常
- [ ] 实时预览正确渲染
- [ ] 自动保存功能正常
