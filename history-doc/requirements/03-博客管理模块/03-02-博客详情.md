# 03-02-博客详情页

## 3.2.1 功能概述

博客详情页展示单篇文章的完整内容，提供良好的阅读体验。

## 3.2.3 功能详情

### 文章信息

| 字段 | 说明 |
|------|------|
| 标题 | 文章标题，最多 200 字符 |
| 分类 | 显示分类名称（可点击跳转分类页） |
| 发布日期 | 格式：YYYY-MM-DD |
| 阅读时间 | 估算阅读时间（字数/200） |
| 阅读量 | 页面加载时 +1 |
| 标签 | 显示标签列表（可点击跳转标签页） |

### 文章内容

| 字段 | 说明 |
|------|------|
| 内容格式 | Markdown |
| 代码高亮 | 支持常见语言（JS, Java, Python, SQL 等） |
| 目录导航 | 根据 H2/H3 自动生成，可点击跳转 |
| 图片 | 支持懒加载，响应式显示 |

### 相关推荐

| 字段 | 说明 |
|------|------|
| 数量 | 3 篇 |
| 选择规则 | 同分类的最新文章，排除当前文章 |

### 阅读量统计

- 同一会话中重复访问不累计
- 刷新页面不累计
- 离开后重新访问累计

## 3.2.4 SEO 元数据

| 标签 | 内容 |
|------|------|
| title | `{文章标题} - {网站名}` |
| description | `{文章摘要}` |
| og:title | `{文章标题}` |
| og:description | `{文章摘要}` |
| og:type | article |
| og:url | `https://yoursite.com/blog/{slug}` |
| article:published_time | ISO 8601 日期 |
| article:modified_time | ISO 8601 日期 |
| article:author | 作者名称 |
| article:section | 分类名称 |

## 3.2.5 用户交互

| 交互 | 行为 |
|------|------|
| 点击目录项 | 平滑滚动到对应章节 |
| 点击标签 | 跳转至 `/blog?tag={tagSlug}` |
| 点击分类 | 跳转至 `/blog?category={categorySlug}` |
| 点击相关文章 | 跳转至对应文章详情页 |
| 滚动阅读 | 显示阅读进度条 |
