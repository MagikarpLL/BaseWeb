# 05-01-错误处理

> 对应 Requirement: [05-01-错误处理.md](../requirements/05-错误处理模块/05-01-错误处理.md)

## 5.1.1 设计原则

| 原则 | 说明 |
|------|------|
| 用户友好 | 错误信息清晰，不显示技术细节 |
| 可恢复 | 提供重试/返回等恢复路径 |
| 静默降级 | 非关键错误不阻塞用户操作 |

---

## 5.1.2 错误分类与处理

### 网络错误

| 场景 | 处理方式 |
|------|----------|
| 断网 | 全局 Toast "网络连接已断开"，恢复后 Toast "网络已恢复" |
| 请求超时（>10s） | Toast "请求超时，请稍后重试" |
| DNS 解析失败 | Toast "网络异常，请检查网络连接" |

### 认证错误

| 错误码 | 场景 | 处理方式 |
|--------|------|----------|
| 40101 | 未提供 Token | 跳转登录页 |
| 40102 | Token 无效或过期 | 自动刷新 Token |
| 40103 | Refresh Token 过期 | 跳转登录页，Toast "登录已过期" |
| 40104 | 账户被锁定 | 显示锁定提示和剩余时间 |

### 业务错误

| 错误码 | 场景 | 处理方式 |
|--------|------|----------|
| 40001 | 参数校验失败 | 高亮字段，显示具体校验信息 |
| 40401 | 文章不存在 | 跳转 404 页面 |
| 40901 | slug 重复 | 高亮 slug 字段，提示"链接地址已被使用" |

### 服务端错误

| 错误码 | 场景 | 处理方式 |
|--------|------|----------|
| 50001 | 服务器内部错误 | Toast "服务器异常，请稍后重试" |
| 50002 | 数据库操作失败 | Toast "操作失败，请稍后重试" |

## 5.1.3 前台错误处理规格

### 页面级错误

| 页面 | 错误场景 | 处理方式 |
|------|----------|----------|
| 首页 | API 失败 | 显示空状态 + 重试按钮，不阻塞骨架屏 |
| 博客列表 | 加载失败 | 显示"加载失败，点击重试" |
| 博客详情 | slug 无效 | `router.replace('/404')` |
| 博客详情 | API 失败 | 显示错误提示 + 重试按钮 |
| 关于页 | API 失败 | 同首页，显示空状态 + 重试 |
| 工具集 | API 失败 | 显示空状态 + 重试 |

### 操作级错误

| 操作 | 错误场景 | 处理方式 |
|------|----------|----------|
| 评论提交 | 网络失败 | Toast "网络异常，请稍后重试"，保留表单内容 |
| 评论提交 | 验证码错误 | 高亮验证码字段，提示"验证码错误" |
| 评论提交 | 内容校验失败 | 高亮对应字段，显示具体校验提示 |

### 全局错误

| 错误 | 处理方式 |
|------|----------|
| JS 运行时错误 | 错误边界捕获，显示"页面出了点问题" + 刷新按钮 |
| 404 | 自定义 404 页面 |

---

## 5.1.4 管理后台错误处理规格

### 表单操作错误

| 操作 | 错误场景 | 处理方式 |
|------|----------|----------|
| 文章保存 | 网络失败 | Toast "保存失败，请检查网络"，保留编辑内容 |
| 文章保存 | Token 过期 | 静默刷新后重试，刷新失败跳转登录 |
| 文章保存 | 参数校验失败 | 高亮错误字段，显示校验提示 |
| 文章删除 | 失败 | Toast 显示错误信息 |
| 配置保存 | 网络失败 | Toast，保留表单数据 |
| 图片上传 | 文件过大 | Toast "文件大小超过限制（最大 5MB）" |
| 图片上传 | 格式不支持 | Toast "不支持的文件格式（仅 JPG/PNG/GIF/WebP）" |
| 图片上传 | 超时 | Toast "上传超时，请重试" |

### 列表页错误

| 场景 | 处理方式 |
|------|----------|
| 列表加载失败 | 显示错误状态 + 重试按钮 |
| 删除成功 | Toast "删除成功" + 刷新列表 |
| 操作失败 | Toast 显示具体错误信息 |

### 自动保存错误

| 场景 | 处理方式 |
|------|----------|
| 自动保存失败 | Toast "自动保存失败"，内容不丢失 |
| 自动保存成功 | 静默处理 |
| 恢复草稿 | 进入编辑页检测到草稿时询问"是否恢复？" |

---

## 5.1.5 加载状态规格

| 状态 | 表现 |
|------|------|
| 首次加载 | 骨架屏（列表页）或居中 loading（详情页） |
| 翻页/筛选 | 列表区域局部 loading，不刷新页面 |
| 表单提交 | 按钮显示 loading + 禁用，防止重复提交 |
| 图片加载 | 渐显效果 + 失败时显示占位图 |

## 5.1.6 前端错误处理代码示例

### Axios 拦截器

```typescript
// 响应拦截器
api.interceptors.response.use(
  (response) => response,
  async (error) => {
    const { status, data } = error.response
    const { code, message } = data

    if (status === 401) {
      if (code === 40102) {
        // Token 过期，尝试刷新
        try {
          await refreshToken()
          return retryRequest(error.config)
        } catch {
          router.push('/login')
          showToast('登录已过期，请重新登录')
        }
      } else if (code === 40103) {
        // Refresh Token 过期
        router.push('/login')
        showToast('登录已过期，请重新登录')
      } else if (code === 40104) {
        // 账户锁定
        showLockDialog(message, remainingTime)
      }
    } else if (status === 404 && code === 40401) {
      router.push('/404')
    } else if (status === 409) {
      // 表单冲突错误，抛给调用方处理
      return Promise.reject({ ...data, isFormError: true })
    } else if (status >= 500) {
      showToast(message || '服务器异常，请稍后重试')
    } else {
      showToast(message || '操作失败')
    }

    return Promise.reject(data)
  }
)
```

### 网络状态监听

```typescript
window.addEventListener('online', () => {
  showToast('网络已恢复', 'success')
})

window.addEventListener('offline', () => {
  showToast('网络连接已断开', 'warning')
})
```

## 5.1.7 验收标准

- [ ] 网络断开时显示全局提示，恢复后自动消失
- [ ] API 返回 4xx 业务错误时，前端显示具体错误信息
- [ ] API 返回 5xx 错误时，显示通用错误提示
- [ ] 文章 slug 无效时重定向到 404 页面
- [ ] 评论提交失败保留表单内容
- [ ] 管理后台表单保存失败不丢失编辑内容
- [ ] Token 过期自动刷新，用户无感知
- [ ] 刷新失败跳转登录页并显示提示
- [ ] 图片上传超限/格式错误有明确 Toast 提示
- [ ] 列表页加载失败显示重试按钮
- [ ] 所有表单提交按钮在请求中禁用防止重复
