## 3.3.3 页面渲染规格

### 文章列表页

| 元素 | 渲染规则 |
|------|----------|
| 表格列 | 标题 / 状态 / 分类 / 发布时间 / 阅读量 / 操作 |
| 状态标签 | 草稿：灰色背景，已发布：绿色背景 |
| 搜索框 | 实时搜索，300ms 防抖 |
| 筛选 | 按分类、按状态筛选 |
| 排序 | 按发布时间、阅读量排序 |
| 操作按钮 | 编辑 / 删除 / 置顶 |

### 文章编辑页

| 区域 | 说明 |
|------|------|
| 标题输入 | 单行输入，最多 200 字符，实时显示字符数 |
| 元信息栏 | 分类下拉 + 标签多选 + 封面图上传 |
| 编辑区 | 左侧 Markdown 编辑器，右侧实时预览 |
| 操作栏 | 发布 / 保存草稿 / 预览 按钮 |

### Markdown 编辑器

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 粗体 | `Ctrl+B` | `**text**` |
| 斜体 | `Ctrl+I` | `*text*` |
| 标题 | `Ctrl+1~6` | `# 到 ######` |
| 链接 | `Ctrl+K` | `[text](url)` |
| 代码块 | `Ctrl+`` | ``` ```code``` ``` |
| 图片 | `Ctrl+Shift+I` | `![alt](url)` |

**工具栏按钮**：
- 格式化：粗体、斜体、删除线
- 标题：H1-H6 下拉
- 列表：无序、有序
- 链接：链接、图片
- 代码：行内代码、代码块
- 表格：插入表格
- 预览：切换预览模式

### 自动保存

| 参数 | 值 |
|------|------|
| 间隔 | 30 秒 |
| 存储 | localStorage |
| 恢复提示 | 进入编辑页时检测未保存草稿 |

  const content = {
    id: articleId,
    title: document.querySelector('#title').value,
    content: editor.getMarkdown(),
    savedAt: Date.now()

### slug 自动生成

```typescript
// 从标题生成 slug
function generateSlug(title: string): string {
  return title
    .toLowerCase()
    .replace(/[^\w\s-]/g, '')  // 移除非字母数字
    .replace(/\s+/g, '-')       // 空格替换为连字符
    .replace(/-+/g, '-')        // 多个连字符合并
    .substring(0, 100)
}
```
