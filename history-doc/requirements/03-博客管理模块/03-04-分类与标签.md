# 03-04-分类与标签管理（管理后台）

## 3.4.1 功能概述

分类和标签管理允许管理员对博客文章进行归类管理。

## 3.4.2 分类管理

### 分类字段

| 字段 | 说明 |
|------|------|
| 分类名称 | 最多 20 字符 |
| slug | URL 友好格式，如 "frontend" |
| 描述 | 分类描述，最多 200 字符 |
| 排序权重 | 越小越靠前 |
| 文章数量 | 该分类下的文章数量（计算字段） |

### 分类列表页

```
┌─────────────────────────────────────────────────────────────┐
│  分类管理                                      [+ 新建分类]   │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────┐   │
│  │  前端        frontend      20篇    [编辑] [删除]      │   │
│  ├─────────────────────────────────────────────────────┤   │
│  │  后端        backend      15篇    [编辑] [删除]      │   │
│  ├─────────────────────────────────────────────────────┤   │
│  │  架构        architecture  10篇    [编辑] [删除]      │   │
│  └─────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
```

### 分类操作

| 操作 | 说明 |
|------|------|
| 新建分类 | 弹出 Modal，填写 name/slug/description |
| 编辑分类 | 弹出 Modal，修改字段 |
| 删除分类 | 确认弹窗，需处理已关联文章（转移分类或强制删除） |
| 排序 | 拖拽调整顺序 |

### slug 自动生成规则

- 中文名称 → 拼音转换
- 空格 → 连字符
- 大写 → 小写
- 特殊字符 → 移除

## 3.4.3 标签管理

### 标签字段

| 字段 | 说明 |
|------|------|
| 标签名称 | 最多 15 字符 |
| slug | URL 友好格式 |
| 文章数量 | 该标签下的文章数量（计算字段） |

### 标签列表页

```
┌─────────────────────────────────────────────────────────────┐
│  标签管理                                      [+ 新建标签]  │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  ┌─────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐             │
│  │ Vue.js  │ │JavaScript│ │ TypeScript│ │ React  │  ...      │
│  │  10篇   │ │  15篇   │ │   8篇   │ │   5篇   │             │
│  └─────────┘ └─────────┘ └─────────┘ └─────────┘             │
│                                                              │
│  [编辑] [删除] 操作按钮在悬停时显示                            │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 标签操作

| 操作 | 说明 |
|------|------|
| 新建标签 | 弹出 Modal，填写 name |
| 编辑标签 | 弹出 Modal，修改 name |
| 删除标签 | 确认弹窗，自动解除与文章的关联 |

## 3.4.4 验收标准

- [ ] 管理员可以创建新分类
- [ ] 管理员可以编辑分类信息
- [ ] 管理员可以删除分类（需处理关联文章）
- [ ] 管理员可以调整分类排序
- [ ] 管理员可以创建新标签
- [ ] 管理员可以编辑标签名称
- [ ] 管理员可以删除标签（自动解除关联）
- [ ] 分类 slug 自动生成
- [ ] 文章数量统计正确
