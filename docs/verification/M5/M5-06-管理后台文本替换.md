# M5-06 - 管理后台文本替换 + Element Plus 中文 - 验证文档

## 验收标准

1. ✅ 管理后台所有界面文本已使用 t() 函数替换
2. ✅ Element Plus 组件在管理后台显示中文（locale: zhCn）
3. ✅ 所有页面标题、按钮文字、提示消息已翻译
4. ✅ Settings 页面的语言设置 Tab 已正确实现

## 测试用例

### 测试 1: Dashboard 页面文本验证

**步骤**:
1. 打开浏览器访问管理后台 `/admin`
2. 检查 Dashboard 页面所有文本是否使用翻译

**预期结果**:
- "Dashboard" 标题 → `t('admin.dashboard')`

## 验证方法

```bash
# 1. 启动前端开发服务器
cd frontend && npm run dev

# 2. 访问 http://localhost:5173/admin
# 3. 检查 Dashboard 页面的翻译是否正确显示

# 4. 访问 http://localhost:5173/admin/posts
# 5. 检查 PostList 页面的翻译是否正确显示

# 6. 切换 Settings 页面的 Language Tab
# 7. 检查语言切换功能是否正常工作
```

## 静态检查

```bash
# 使用 TypeScript 编译器检查类型错误
cd frontend && npx tsc --noEmit

# 检查是否所有 admin 视图都引入了 useLocale
grep -r "useLocale" frontend/src/views/admin/
```
