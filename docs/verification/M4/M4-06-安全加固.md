# M4-06 - 安全加固

> 对应任务: [01-迭代计划.md](../../plan/01-迭代计划.md#m4-06-安全加固rate-limit-cors-xss-csrf)
> 涉及 Spec: [04-认证与安全.md](../../spec/04-认证与安全.md)

## 验证目标

Rate Limit、CORS、XSS、CSRF 防护正常。

## 测试用例

### TC-01: Rate Limit

**前置条件：**
- 无

**测试步骤：**
1. 快速连续发送 100+ 请求到同一 API
2. 验证被限流

**预期结果：**
- 返回 429 Too Many Requests
- 提示限流信息

### TC-02: CORS 配置

**前置条件：**
- 无

**测试步骤：**
1. 从不同域名发送 API 请求
2. 验证 CORS 头

**预期结果：**
- 正确配置 Access-Control-Allow-Origin
- 预检请求处理正确

### TC-03: XSS 防护

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 提交包含 `<script>alert(1)</script>` 的内容
2. 验证内容被转义

**预期结果：**
- JavaScript 不执行
- 内容被 HTML 转义

### TC-04: CSRF 防护

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 检查表单提交是否需要 CSRF Token
2. 验证 Token 验证

**预期结果：**
- 表单包含 CSRF Token
- 无 Token 请求被拒绝
