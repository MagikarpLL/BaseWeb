# 02-03-工具集页面

> 对应 Requirement: [02-03-工具集页面.md](../requirements/02-个人网站展示模块/02-03-工具集页面.md)

## 2.3.1 设计原则

| 原则 | 说明 |
|------|------|
| 即时可用 | 工具操作无需页面刷新，实时输出结果 |
| 分类清晰 | 工具按功能分类，支持快速筛选 |
| 可扩展 | 工具配置化，易于新增工具 |

---

## 2.3.2 数据模型

### 工具列表 API

**接口**: `GET /api/public/tools`

```json
{
  "code": 200,
  "data": {
    "categories": ["all", "formatter", "encoder", "converter", "generator", "other"],
    "tools": [
      {
        "id": 1,
```

## 2.3.3 页面渲染规格

### 工具列表页

| 元素 | 渲染规则 |
|------|----------|
| 页面标题 | 居中，字号 `2rem`，上方留白 `80px` |
| 分类导航 | 水平滚动，`active` 状态高亮，底部边框 `#667eea` |
| 工具卡片 | 网格布局，`3` 列（桌面）/ `2` 列（平板）/ `1` 列（手机） |
| 卡片高度 | 统一 `180px`，内容靠上 |
| 无工具状态 | 显示"该分类下暂无工具" |

**分类导航交互**：
- 点击分类 → 筛选该分类工具
- `all` 分类显示所有工具
- 切换分类无页面刷新，局部更新

### 工具详情页

| 区域 | 说明 |
|------|------|
| 标题区 | 工具名称 + 图标 + 描述 |
| 输入区 | 文本框/文本域，`100%` 宽度，`300px` 高度 |
| 操作区 | 功能按钮，水平排列 |
| 输出区 | 结果显示区，同输入区尺寸 |

**操作按钮样式**：
- 背景：`#667eea`，圆角 `8px`
- Hover：背景加深 `10%`
- Active：缩放 `0.95`
- Loading：` spinner` 图标 + 禁用状态

## 2.3.4 内置工具规格

### 2.3.4.1 JSON 格式化

| 功能 | 输入 | 输出 | 说明 |
|------|------|------|------|
| 格式化 | `{"a":1}` | 格式化后 JSON | 2 空格缩进 |
| 压缩 | `{"a":1}` | `{"a":1}` | 移除空格 |
| 校验 | `{"a":1}` | `✓ 有效 JSON` 或错误信息 | 语法检查 |

### 2.3.4.2 URL 编码

| 功能 | 输入 | 输出 |
|------|------|------|
| 编码 | `hello world` | `hello%20world` |
| 解码 | `hello%20world` | `hello world` |

### 2.3.4.3 Base64 编解码

| 功能 | 输入 | 输出 |
|------|------|------|
| 编码 | `hello` | `aGVsbG8=` |
| 解码 | `aGVsbG8=` | `hello` |

### 2.3.4.4 时间戳转换

| 功能 | 输入 | 输出 |
|------|------|------|
| 时间→戳 | `2026-04-20 10:30:00` | `1742527800000` |
| 戳→时间 | `1742527800000` | `2026-04-20 10:30:00` |

### 2.3.4.5 密码生成器

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 长度 | `16` | 8-128 字符 |
| 大写 | `true` | 包含 A-Z |
| 小写 | `true` | 包含 a-z |
| 数字 | `true` | 包含 0-9 |
| 特殊字符 | `false` | 包含 !@#$% |

### 2.3.4.6 UUID 生成器

| 功能 | 输出格式 |
|------|----------|
| 生成 UUID | `xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx` |

### 2.3.4.7 颜色转换

| 输入格式 | 示例 |
|----------|------|
| HEX | `#667eea` |
| RGB | `rgb(102, 126, 234)` |
| HSL | `hsl(230, 70%, 66%)` |

### 2.3.4.8 Cron 表达式

| 功能 | 说明 |
|------|------|
| 解析 | 显示下次 5 个执行时间 |
| 验证 | 检查表达式是否有效 |
