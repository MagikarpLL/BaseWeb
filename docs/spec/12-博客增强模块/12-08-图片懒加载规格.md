# 12-08-图片懒加载规格

> 对应 Requirement: [12-08-图片懒加载.md](../requirements/12-博客增强模块/12-08-图片懒加载.md)

## 12.8.1 概述 / 设计原则

|| 原则 | 说明 |
||------|------|
|| Intersection Observer | 使用原生 API，性能优良 |
|| 渐进式加载 | 占位图 + 淡入效果 |
|| 降级处理 | 不支持时正常加载所有图片 |

## 12.8.2 具体规格

### 12.8.2.1 自定义指令

```typescript
// directives/lazyload.ts
import type { Directive, DirectiveBinding } from 'vue'

const lazyload: Directive = {
  mounted(el: HTMLElement, binding: DirectiveBinding) {
    if (!('IntersectionObserver' in window)) {
      // 降级：直接设置 src
      if (binding.value) {
        (el as HTMLImageElement).src = binding.value
      }
      return
    }

    const observer = new IntersectionObserver(
      (entries) => {
        entries.forEach((entry) => {
          if (entry.isIntersecting) {
            const img = entry.target as HTMLImageElement
            const src = binding.value
            if (src) {
              img.src = src
              img.classList.add('loaded')
            }
            observer.unobserve(img)
          }
        })
      },
      {
        rootMargin: '50px 0px',
        threshold: 0.01
      }
    )

    observer.observe(el)
  }
}

export default lazyload
```

### 12.8.2.2 注册指令

```typescript
// main.ts
import lazyload from '@/directives/lazyload'

app.directive('lazyload', lazyload)
```

### 12.8.2.3 使用方式

```vue
<!-- 文章内容图片 -->
<img v-lazyload="post.imageUrl" class="lazy" alt="..." />

<!-- 相关文章缩略图 -->
<img v-lazyload="relatedPost.coverImage" class="lazy" alt="..." />
```

### 12.8.2.4 CSS 样式

```css
/* 占位图样式 */
.lazy {
  opacity: 0;
  transition: opacity 0.3s ease;
  background-color: #f5f5f5;
  min-height: 100px;
}

/* 占位图加载中 */
.lazy:not([src]) {
  background-image: url('data:image/svg+xml,...');  /* loading SVG */
  background-repeat: no-repeat;
  background-position: center;
}

/* 加载完成 */
.lazy.loaded {
  opacity: 1;
}

/* 图片加载完成后的过渡 */
img[lazyload] {
  background: linear-gradient(90deg, #f5f5f5 25%, #eaeaea 50%, #f5f5f5 75%);
  background-size: 200% 100%;
  animation: shimmer 1.5s infinite;
}

@keyframes shimmer {
  0% { background-position: 200% 0; }
  100% { background-position: -200% 0; }
}
```

### 12.8.2.5 影响范围

|| 位置 | 处理方式 |
||------|-----------|
|| 文章内容 `<img>` | 使用 `v-lazyload` 指令 |
|| 封面图 `<img>` | 正常加载（首屏关键图片） |
|| 相关文章缩略图 | 使用 `v-lazyload` 指令 |
|| 作者头像 | 正常加载（小图片） |
|| 懒加载占位图 | 内联 SVG 或纯色 |

### 12.8.2.6 Markdown 渲染适配

在渲染 Markdown 内容时，给所有 `<img>` 添加 `v-lazyload` 指令：

```typescript
// 创建自定义 marked 渲染器
const renderer = new marked.Renderer()

// 处理图片
renderer.image = function({ href, title, text }) {
  return `<img v-lazyload="${href}" class="lazy" alt="${text}" title="${title || ''}" />`
}

marked.use({ renderer })
```

## 12.8.3 验收标准

- [ ] 文章内图片实现懒加载
- [ ] 滚动到图片位置时才加载
- [ ] 加载时有占位图显示
- [ ] 图片加载完成后有淡入效果
- [ ] 封面图正常加载（不懒加载）
- [ ] 相关文章缩略图懒加载
- [ ] 浏览器不支持 Intersection Observer 时正常降级
- [ ] 页面性能指标有提升（特别是 LCP）
