# 03-01-博客列表页

## 3.1.1 功能概述

博客列表页展示所有已发布的文章，支持分页浏览和分类筛选。

## 3.1.3 功能详情

### 筛选功能

| 筛选维度 | 选项 | 说明 |
|----------|------|------|
| 分类 | 下拉选择 | 显示所有分类 + "全部分类"选项 |
| 标签 | 下拉选择 | 显示所有标签 + "全部标签"选项 |
| 排序 | 下拉选择 | 最新发布 / 最新更新 / 阅读量最高 |

### 分页功能

| 字段 | 说明 |
|------|------|
| 每页数量 | 10 篇文章 |
| 显示范围 | 显示当前页前后各 2 页的页码 |
| 总页数 | 根据文章总数计算 |

### 文章卡片信息

| 字段 | 说明 |
|------|------|
| 标题 | 文章标题，最多 100 字符 |
| 摘要 | 文章摘要，最多 200 字符 |
| 发布日期 | 格式：YYYY-MM-DD |
| 阅读时间 | 根据内容长度估算：字数/200（字/分钟） |
| 分类 | 显示分类名称 |
| 标签 | 显示标签列表（最多显示 3 个） |
| 阅读量 | 显示数字 |

## 3.1.4 SEO 元数据

| 标签 | 内容 |
|------|------|
| title | `博客 - {网站名}` |
| description | `{网站名}的技术博客，分享前端、后端、架构等领域的经验和见解` |
| og:title | `博客 - {网站名}` |
| og:type | website |

## 3.1.5 用户交互

| 交互 | 行为 |
|------|------|
| 点击文章卡片 | 跳转至 `/blog/:slug` |
| 选择分类 | 筛选该分类下的文章 |
| 选择标签 | 筛选该标签下的文章 |
| 选择排序 | 按选择重新排序文章列表 |
| 点击分页 | 加载对应页的文章 |
