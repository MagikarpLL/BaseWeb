# 12-03-社交分享优化

## 12.3.1 功能概述

完善博客详情页的社交分享元数据（Open Graph 和 Twitter Card），提升分享到社交平台时的预览效果。

## 12.3.2 页面布局

```
┌─────────────────────────────────────────────────────────────┐
│                        Header                                │
└─────────────────────────────────────────────────────────────┘
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│   分类  ·  发布日期  ·  阅读时间  ·  阅读量                    │
│                                                              │
│                      文章标题                                 │
│                                                              │
│   标签1  标签2  标签3                                          │
│                                                              │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│                      文章内容                                 │
│                                                              │
│   ┌─────────────────────────────────────────────────────┐   │
│   │  目录导航                                           │   │
│   └─────────────────────────────────────────────────────┘   │
│                                                              │
├─────────────────────────────────────────────────────────────┤
│  分享到:  [X] [微博] [QQ] [微信] [复制链接]                   │
└─────────────────────────────────────────────────────────────┘
```

## 12.3.3 功能详情

### 12.3.3.1 Open Graph 元数据

|| 标签 | 内容 | 当前状态 |
||------|------|----------|
| og:title | 文章标题 | ✅ 已有 |
| og:description | 文章摘要 | ✅ 已有 |
| og:image | 封面图 URL | ✅ 已有 |
| og:url | 页面完整 URL | ❌ 缺失 |
| og:type | article | ✅ 已有 |
| article:published_time | ISO 8601 发布时间 | ✅ 已有 |
| article:modified_time | ISO 8601 修改时间 | ✅ 已有 |
| article:author | 作者名称 | ✅ 已有 |
| article:section | 分类名称 | ❌ 缺失 |
| article:tag | 标签列表 | ❌ 缺失 |

### 12.3.3.2 Twitter Card 元数据

|| 标签 | 内容 | 当前状态 |
||------|------|----------|
| twitter:card | summary_large_image 或 summary | ✅ 已有 |
| twitter:title | 文章标题 | ✅ 已有 |
| twitter:description | 文章摘要 | ✅ 已有 |
| twitter:image | 封面图 URL | ✅ 已有 |
| twitter:site | @用户名（可选） | ❌ 缺失 |

### 12.3.3.3 JSON-LD 结构化数据

```json
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "文章标题",
  "description": "文章摘要",
  "image": "封面图 URL",
  "datePublished": "2026-05-29T10:00:00Z",
  "dateModified": "2026-05-29T10:00:00Z",
  "author": {
    "@type": "Person",
    "name": "作者名"
  },
  "publisher": {
    "@type": "Organization",
    "name": "网站名"
  },
  "mainEntityOfPage": {
    "@type": "WebPage",
    "@id": "https://yoursite.com/blog/article-slug"
  }
}
```

### 12.3.3.4 分享平台要求

|| 平台 | 图片尺寸要求 | 说明 |
||------|-------------|------|
|| Facebook | 1200x630 | 最小 600x315 |
|| Twitter | 1200x675 | 最小 300x157 |
|| 微博 | 720x480 | - |

> 社交分享优化涉及的前端 meta 标签实现见 Spec: [12-03-社交分享优化规格.md](../../spec/12-博客增强模块/12-03-社交分享优化规格.md)

## 12.3.4 验收标准

- [ ] og:url 正确设置为页面完整 URL
- [ ] article:section 包含分类名称
- [ ] article:tag 包含所有标签
- [ ] twitter:site 正确设置（如有 Twitter 账号）
- [ ] JSON-LD Article 结构完整
- [ ] og:image 尺寸符合要求（1200x630 或更大）
- [ ] 分享到 Twitter 时预览效果正确
- [ ] 分享到 Facebook 时预览效果正确
