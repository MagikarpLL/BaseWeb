## 命名规范

### Controller 命名

| 类型 | 规范 | 示例 |
|------|------|------|
| Controller 类 | `XxxController` | `PostController` |
| Controller 方法 | REST 风格 | `GET /api/blog/posts` |
| URL 变量 | `camelCase` | `/api/blog/posts/{postId}` |

### Service 命名

| 类型 | 规范 | 示例 |
|------|------|------|
| 接口 | `XxxService` | `PostService` |
| 实现类 | `XxxServiceImpl` | `PostServiceImpl` |
| 方法 | 动宾短语 | `getPostBySlug()` |

### Mapper 命名

| 类型 | 规范 | 示例 |
|------|------|------|
| 接口 | `XxxMapper` | `PostMapper` |
| XML 文件 | `XxxMapper.xml` | `PostMapper.xml` |

### Entity 命名

| 类型 | 规范 | 示例 |
|------|------|------|
| 类名 | 与表名对应，大写开头 | `BlogPost` → 表 `blog_post` |
| 字段 | camelCase | `publishedAt` → 列 `published_at` |
| 主键 | `id` | - |
| 外键 | `xxxId` | `authorId` → 列 `author_id` |

## 包职责说明

| 包 | 职责 |
|-----|------|
| `config` | 所有配置类（Security、CORS、MyBatis 等） |
| `controller` | 接收 HTTP 请求，参数校验，调用 Service |
| `service` | 业务逻辑处理，事务管理 |
| `mapper` | 数据库操作，SQL 执行 |
| `model/entity` | 数据库表对应的 Java 实体 |
| `model/dto` | 数据传输对象，分 request/response |
| `exception` | 异常定义和全局处理 |
| `security` | JWT 认证、过滤器、Token 管理 |

---

## 层级调用关系

```
HTTP Request
    ↓
Controller (参数校验)
    ↓
Service (业务逻辑)
    ↓
Mapper (数据库操作)
    ↓
Database
```

## 统一响应格式

```java
public record ApiResponse<T>(
    int code,       // 状态码: 200=成功, 400=参数错误, 401=未认证, 404=不存在, 500=服务器错误
    String message, // 消息: "Success" 或错误描述
    T data,         // 数据体
    long timestamp  // 时间戳
) {
    public static <T> ApiResponse<T> success(T data) {
        return new ApiResponse<>(200, "Success", data, System.currentTimeMillis());
    }

    public static <T> ApiResponse<T> error(int code, String message) {
        return new ApiResponse<>(code, message, null, System.currentTimeMillis());
    }
}
```
