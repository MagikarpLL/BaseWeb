# 08-01-评论功能

## 8.1.1 功能概述

访客可以在博客文章下方发表评论，管理员可以在后台审核和管理评论。

## 8.1.2 评论状态

| 状态 | 说明 |
|------|------|
| pending | 待审核 |
| approved | 已通过 |
| rejected | 已拒绝 |

## 8.1.3 功能详情

### 访客评论

| 字段 | 规则 |
|------|------|
| 昵称 | 必填，2-20 字符 |
| 邮箱 | 必填，有效邮箱格式 |
| 评论内容 | 必填，5-2000 字符 |
| 验证码 | 必填，防止机器人 |

### 评论显示规则

- 只显示 `approved` 状态的评论
- 按时间正序排列
- 支持嵌套回复（最多 2 层）
- @ 提及显示为链接

### 管理员评论管理

| 功能 | 说明 |
|------|------|
| 评论列表 | 分页显示，可筛选状态 |
| 审核评论 | 批准/拒绝 |
| 回复评论 | 以管理员身份回复 |
| 删除评论 | 永久删除 |

### 评论筛选

| 条件 | 选项 |
|------|------|
| 状态 | 待审核/已通过/已拒绝 |
| 文章 | 按文章筛选 |
| 时间范围 | 按日期筛选 |

## 8.1.5 验证码

- 4 位随机字符
- 10 分钟有效期
- 验证失败可刷新

## 8.1.6 验收标准

- [ ] 访客可以提交评论
- [ ] 评论需审核后才能显示
- [ ] 支持嵌套回复（2 层）
- [ ] 管理员可以审核评论
- [ ] 管理员可以回复评论
- [ ] 管理员可以删除评论
- [ ] 评论列表支持分页
- [ ] 验证码防止机器人
- [ ] 评论内容支持 Markdown
