## 3.2.3 页面渲染规格

### 文章头部

| 元素 | 渲染规则 |
|------|----------|
| 标题 | 字号 `2.5rem`，加粗，颜色 `#333`，居中 |
| 元信息栏 | 分类 · 发布日期 · 阅读时间 · 阅读量，水平排列，间距 `8px` |
| 标签 | 胶囊样式，hover 下划线，可点击跳转 `/blog?tag={slug}` |
| 封面图 | 可选，全宽，高度 `400px`，`object-fit: cover` |

### 文章内容

| 元素 | 渲染规则 |
|------|----------|
| Markdown 渲染 | 支持 `h1-h6`/`p`/`ul`/`ol`/`blockquote`/`table` |
| 代码块 | 语法高亮（highlight.js），主题 `github-dark`，圆角 `8px` |
| 图片 | 响应式 `max-width: 100%`，懒加载 |
| 引用 | 左边框 `4px solid #667eea`，背景 `#f8f9fa` |
| 表格 | 斑马纹，横向滚动支持 |

**内容区样式**：
- 最大宽度 `800px`，居中
- 行高 `1.8`
- 段落间距 `1.5em`

### 目录导航

| 元素 | 渲染规则 |
|------|----------|
| 标题提取 | 从 `h2`/`h3` 生成 |
| 显示位置 | 右侧固定，`200px` 宽 |
| 激活状态 | 当前滚动位置对应的目录项高亮 |
| 移动端 | 隐藏目录，文章内提供锚点跳转 |

### 作者信息

| 元素 | 渲染规则 |
|------|----------|
| 头像 | `48x48px` 圆形 |
| 姓名 | 加粗 |
| 发布日期 | 格式 `YYYY-MM-DD` |

### 相关推荐

| 元素 | 渲染规则 |
|------|----------|
| 数量 | `3` 篇 |
| 选择规则 | 同分类的最新文章，排除当前文章 |
| 卡片样式 | 小卡片，缩略图 + 标题 |

## 3.2.5 SEO 与结构化数据

### Meta 标签

| 标签 | 内容 |
|------|------|
| title | `{文章标题} - {网站名}` |
| description | `{文章摘要}` |
| og:title | `{文章标题}` |
| og:description | `{文章摘要}` |
| og:type | article |
| og:url | `https://yoursite.com/blog/{slug}` |
| article:published_time | ISO 8601 |
| article:modified_time | ISO 8601 |
| article:author | 作者名 |
| article:section | 分类名 |

### JSON-LD 结构化数据

```json
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "文章标题",
  "description": "文章摘要",
  "datePublished": "2026-04-20",
  "dateModified": "2026-04-21",
  "author": { "@type": "Person", "name": "张三" },
  "image": "封面图 URL"
}
```
