# M1-04 - 数据库 Schema + 初始化数据

> 对应任务: [01-迭代计划.md](../../plan/01-迭代计划.md#m1-04-数据库-schema-初始化数据)
> 涉及 Spec: [03-数据库设计.md](../../spec/03-数据库设计.md)

## 验证目标

数据库 Schema 创建成功，初始化数据正确。

## 测试用例

### TC-01: Schema 创建验证

**前置条件：**
- PostgreSQL 已启动

**测试步骤：**
1. 执行数据库迁移脚本或初始化 SQL
2. 连接数据库 `psql -U postgres -d personal_website`
3. 执行 `\dt` 列出所有表

**预期结果：**
- 表已创建：users, posts, categories, tags, comments 等
- 表结构符合 `03-数据库设计.md` 规范

### TC-02: 索引验证

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 执行 `\di` 列出所有索引

**预期结果：**
- 索引已创建（posts_slug_idx, posts_category_idx 等）

### TC-03: 初始化数据验证

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 查询初始化数据 `SELECT * FROM users;`
2. 查询分类 `SELECT * FROM categories;`

**预期结果：**
- 存在默认管理员账户
- 存在默认分类

### TC-04: 外键约束验证

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 尝试删除有关联数据的分类
2. 验证删除失败或级联删除

**预期结果：**
- 外键约束正常工作
