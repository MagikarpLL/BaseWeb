# 09-01-访问统计

> 对应 Requirement: [09-01-访问统计.md](../requirements/09-访问统计模块/09-01-访问统计.md)

## 9.1.1 设计原则

| 原则 | 说明 |
|------|------|
| 实时性 | 今日数据实时更新 |
| 隐私合规 | 不记录个人敏感信息，IP 仅用于地区统计 |
| 性能优先 | 埋点不影响页面加载 |

**接口**: `GET /api/admin/analytics/trend`

## 9.1.3 管理后台统计面板渲染规格

### 数据概览卡片

| 指标 | 卡片样式 |
|------|----------|
| 今日 PV/UV | 大号数字，字号 `2rem`，加粗 |
| 昨日 PV/UV | 同上 |
| 较昨日变化 | 绿色（上涨）/红色（下跌）+ 百分比 |
| 当前在线 | 数字 + "人在线" |

### 趋势图

| 元素 | 规格 |
|------|------|
| 图表类型 | 折线图（ECharts） |
| X 轴 | 日期 |
| Y 轴 | PV/UV |
| 图例 | PV（蓝色）/ UV（绿色） |
| 交互 | hover 显示具体数值 |
| 切换 | 7 天 / 30 天 Tab |

### 热门页面

| 元素 | 规格 |
|------|------|
| 列表 | TOP 10，每行显示排名/页面/PV |
| 排序 | 按 PV 降序 |
| 点击 | 跳转该页面 |

### 来源分布

| 元素 | 规格 |
|------|------|
| 图表类型 | 饼图 |
| 标签 | 直接访问 / 搜索引擎 / 外部链接 |
| 颜色 | 蓝 / 绿 / 橙 |

### 设备/浏览器分布

| 元素 | 规格 |
|------|------|
| 图表类型 | 横向柱状图 |
| 显示 | 设备/浏览器名称 + 百分比 |
| 颜色 | 统一主题色 |

## 9.1.4 关键逻辑

### 访客 ID 生成

```typescript
function getVisitorId(): string {
  let visitorId = getCookie('visitor_id')
  if (!visitorId) {
    visitorId = generateUUID()
    setCookie('visitor_id', visitorId, 365) // 1 年有效期
  }
  return visitorId
}
```

### 埋点发送策略

```typescript
// 使用 requestIdleCallback 批量发送
function trackPageView(url: string) {
  const data = {
    type: 'pageview',
    url,
    visitorId: getVisitorId(),
    referrer: document.referrer,
    timestamp: Date.now()
  }

  // 非阻塞发送
  if ('requestIdleCallback' in window) {
    requestIdleCallback(() => sendBeacon(data))
  } else {
    sendBeacon(data)
  }
}

function sendBeacon(data: object) {
  navigator.sendBeacon('/api/analytics/track', JSON.stringify(data))
}
```

### 爬虫过滤

```typescript
function isBot(userAgent: string): boolean {
  const bots = [
    'Googlebot', 'Bingbot', 'Baiduspider',
    'YandexBot', 'Sogou', '360Spider'
  ]
  return bots.some(bot => userAgent.includes(bot))
}
```

## 9.1.5 隐私合规

| 措施 | 说明 |
|------|------|
| IP 处理 | 只用于地区统计，不存储原始 IP |
| Cookie | 仅使用匿名访客 ID |
| GDPR | 不收集个人身份信息 |
| 数据保留 | 90 天内数据详细，90 天后聚合 |
