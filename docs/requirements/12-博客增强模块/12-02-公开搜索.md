# 12-02-公开搜索

## 12.2.1 功能概述

为博客列表页添加公开搜索功能，允许访客通过关键词搜索文章，提升内容可发现性。

## 12.2.3 功能详情

### 12.2.3.1 搜索入口

|| 项目 | 说明 |
||------|------|
| 位置 | 博客列表页标题下方 |
| 输入框占位符 | "搜索文章..." |
| 搜索按钮 | "搜索" 文字按钮 |
| 支持回车 | 输入框支持回车键触发搜索 |

### 12.2.3.2 搜索行为

|| 项目 | 说明 |
||------|------|
| 搜索范围 | 文章标题 + 文章内容 |
| 关键词高亮 | 搜索结果中匹配的文字添加高亮样式 |
| 分页展示 | 搜索结果分页，每页10篇 |
| 清空搜索 | 清空搜索框后恢复全列表 |

### 12.2.3.3 搜索结果展示

|| 项目 | 说明 |
||------|------|
| 结果计数 | "搜索到 **X** 篇相关文章" |
| 关键词显示 | 显示当前搜索的关键词 |
| 空结果 | "未找到相关文章，请尝试其他关键词" |

> 搜索功能的技术实现细节（API 端点、数据库查询）见 Spec: [12-02-公开搜索规格.md](../../spec/12-博客增强模块/12-02-公开搜索规格.md)

## 12.2.4 验收标准

- [ ] 博客列表页有搜索框
- [ ] 输入关键词后回车可以搜索
- [ ] 搜索结果正确匹配标题和内容
- [ ] 搜索结果中关键词高亮显示
- [ ] 搜索结果分页展示
- [ ] 清空搜索框恢复全列表
- [ ] 无搜索结果时显示提示
- [ ] 筛选和搜索可以组合使用
