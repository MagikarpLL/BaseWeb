# M1-16 - 全局布局

> 对应任务: [01-迭代计划.md](../../plan/01-迭代计划.md#m1-16-全局布局header-footer-响应式导航)
> 涉及 Spec: [09-前端路由设计.md](../../spec/09-前端路由设计.md)

## 验证目标

全局布局（Header + Footer + 响应式导航）正常显示。

## 测试用例

### TC-01: Header 显示

**前置条件：**
- 前端服务已启动

**测试步骤：**
1. 访问任意前台页面
2. 验证 Header 显示

**预期结果：**
- Header 正常显示
- 包含网站名称/Logo
- 包含导航菜单（首页、博客、关于、工具集）

### TC-02: 导航链接

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 点击导航"博客"链接
2. 验证跳转

**预期结果：**
- 跳转到 `/blog`

### TC-03: Footer 显示

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 访问任意页面
2. 验证 Footer 显示

**预期结果：**
- Footer 正常显示
- 包含版权信息
- 可能包含社交链接

### TC-04: 响应式 - 桌面端

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 在桌面端窗口访问页面
2. 验证导航显示

**预期结果：**
- 导航菜单完整显示
- 不是汉堡菜单

### TC-05: 响应式 - 移动端

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 缩小窗口到移动端宽度（如 375px）
2. 验证导航变化

**预期结果：**
- 可能变成汉堡菜单
- 点击展开导航菜单

### TC-06: 全局样式一致

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 访问多个不同页面
2. 验证 Header 和 Footer 一致

**预期结果：**
- Header 和 Footer 在所有页面保持一致
- 主题样式统一

## 验收标准

- [ ] TC-06: 全局样式一致通过
- [ ] 所有测试用例通过
