# 05-01-错误处理与边界场景

## 5.1.1 功能概述

本模块定义用户在使用网站过程中可能遇到的错误场景，以及系统应提供的错误反馈和恢复机制。覆盖网络异常、API 错误、表单校验、资源不存在等场景。

## 5.1.2 错误分类

| 错误类型 | 触发场景 | 用户感知 |
|----------|----------|----------|
| 网络错误 | 断网、请求超时（>10s）、DNS 解析失败 | 页面级或操作级错误提示 |
| 认证错误 | Token 过期、无效 Token、无权限访问 | 自动刷新或跳转登录 |
| 业务错误 | 参数校验失败、资源不存在、重复操作 | 表单级或 Toast 提示 |
| 服务端错误 | 500 内部错误、服务不可用 | 通用错误页面 |
| 前端异常 | JS 运行时错误、组件渲染错误 | 错误边界兜底 |

## 5.1.3 前台页面错误处理

### 页面级错误

| 页面 | 错误场景 | 处理方式 |
|------|----------|----------|
| 首页 | API 数据加载失败 | 显示空状态 + 重试按钮，不阻塞页面骨架 |
| 博客列表 | 文章加载失败 | 显示"加载失败，点击重试" |
| 博客详情 | 文章不存在（slug 无效） | 重定向到 404 页面 |
| 博客详情 | 文章加载失败 | 显示错误提示 + 重试按钮 |
| 关于页 | 数据加载失败 | 同首页，显示空状态 + 重试 |
| 工具集 | 工具列表加载失败 | 显示空状态 + 重试 |

### 操作级错误

| 操作 | 错误场景 | 处理方式 |
|------|----------|----------|
| 评论提交 | 网络失败 | Toast "网络异常，请稍后重试"，保留表单内容 |
| 评论提交 | 验证码错误 | 高亮验证码字段，提示"验证码错误" |
| 评论提交 | 内容校验失败 | 高亮对应字段，显示具体校验提示 |
| 评论提交 | 服务器错误 | Toast "提交失败，请稍后重试" |

### 全局错误

| 错误 | 处理方式 |
|------|----------|
| 网络断开 | 全局 Toast "网络连接已断开"，恢复后自动 Toast "网络已恢复" |
| 404 页面 | 显示自定义 404 页面，提供返回首页/博客入口 |
| 未知 JS 错误 | 错误边界捕获，显示"页面出了点问题" + 刷新按钮 |

## 5.1.4 管理后台错误处理

### 表单操作错误

| 操作 | 错误场景 | 处理方式 |
|------|----------|----------|
| 文章保存 | 网络失败 | Toast "保存失败，请检查网络"，保留编辑内容不丢失 |
| 文章保存 | Token 过期 | 静默刷新 Token 后重试，刷新失败则跳转登录 |
| 文章保存 | 参数校验失败 | 高亮错误字段，显示具体校验提示 |
| 文章删除 | 服务器错误 | Toast "删除失败，请稍后重试" |
| 配置保存 | 网络失败 | Toast "保存失败"，不丢弃表单数据 |
| 图片上传 | 文件过大 | Toast "文件大小超过限制（最大 5MB）" |
| 图片上传 | 格式不支持 | Toast "不支持的文件格式（仅 JPG/PNG/GIF/WebP）" |
| 图片上传 | 上传超时 | Toast "上传超时，请重试" |

### 列表页错误

| 错误场景 | 处理方式 |
|----------|----------|
| 列表加载失败 | 显示错误状态 + 重试按钮，不显示空状态 |
| 删除成功 | Toast "删除成功" + 刷新列表 |
| 操作失败 | Toast 显示具体错误信息 |

### 自动保存

| 场景 | 处理方式 |
|------|----------|
| 文章自动保存失败 | Toast "自动保存失败"，编辑器不丢失内容 |
| 自动保存成功 | 静默处理，不弹提示 |
| 恢复草稿 | 进入编辑页时提示"检测到未保存的草稿，是否恢复？" |

## 5.1.5 认证相关错误

| 场景 | 处理方式 |
|------|----------|
| Access Token 过期 | 自动使用 Refresh Token 刷新，用户无感知 |
| Refresh Token 过期 | 跳转登录页，Toast "登录已过期，请重新登录" |
| 并发请求 Token 刷新 | 只触发一次刷新，其他请求排队等待 |
| 登录失败（密码错误） | 表单下方显示"用户名或密码错误" |
| 账户被锁定 | 表单下方显示"账户已锁定，请 15 分钟后重试" |
| 无权限访问管理 API | 跳转登录页 |

## 5.1.6 加载状态

| 状态 | 表现 |
|------|------|
| 首次加载 | 骨架屏（首页/列表页）或居中 loading（详情页） |
| 翻页/筛选 | 列表区域局部 loading，不刷新页面 |
| 表单提交 | 按钮显示 loading 状态 + 禁用，防止重复提交 |
| 图片加载 | 渐显效果 + 失败时显示占位图 |

## 5.1.7 验收标准

- [ ] 网络断开时显示全局提示，恢复后自动消失
- [ ] API 返回业务错误时，前端显示具体错误信息
- [ ] API 返回 500 错误时，显示通用错误提示
- [ ] 文章不存在的 slug 访问，重定向到 404 页面
- [ ] 评论提交失败保留表单内容
- [ ] 管理后台表单保存失败不丢失编辑内容
- [ ] Token 过期自动刷新，用户无感知
- [ ] 刷新失败跳转登录页并提示
- [ ] 图片上传超限/格式错误有明确提示
- [ ] 列表页加载失败显示重试按钮
- [ ] 所有表单提交按钮在请求中禁用防重复
