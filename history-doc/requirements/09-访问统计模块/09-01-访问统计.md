# 09-01-访问统计

## 9.1.1 功能概述

记录网站访问数据，提供 PV/UV 统计、页面访问排行、访客来源分析等功能。

## 9.1.2 统计指标

### 实时统计（当日）

| 指标 | 说明 |
|------|------|
| 今日 PV | 今日页面浏览量 |
| 今日 UV | 今日独立访客数 |
| 当前在线 | 最近 15 分钟有活动的访客 |

### 趋势统计

| 指标 | 说明 |
|------|------|
| 7 天趋势 | 最近 7 天每日 PV/UV |
| 30 天趋势 | 最近 30 天每日 PV/UV |

### 页面统计

| 指标 | 说明 |
|------|------|
| 热门页面 | PV 最高的页面 TOP 10 |
| 页面平均停留时间 | 各页面平均访问时长 |

### 来源统计

| 指标 | 说明 |
|------|------|
| 来源分布 | 直接访问/搜索引擎/外部链接 |
| 搜索引擎 | Google/Bing/Baidu/其他 |
| 设备分布 | PC/移动端 |
| 浏览器分布 | Chrome/Firefox/Safari/其他 |
| 国家/地区分布 | 基于 IP 定位 |

## 9.1.4 隐私合规

- 不记录个人敏感信息
- IP 只用于地区统计，不存储具体 IP
- 遵循 GDPR/中国网络安全法

## 9.1.5 抗干扰

- 过滤爬虫流量（User-Agent 识别）
- 过滤无效访问（短暂停留）

## 9.1.6 验收标准

- [ ] 实时显示今日 PV/UV
- [ ] 显示 7 天/30 天趋势图
- [ ] 显示热门页面 TOP 10
- [ ] 显示来源分布
- [ ] 显示设备/浏览器分布
- [ ] 统计不影响页面加载性能
- [ ] 管理后台有独立统计面板
