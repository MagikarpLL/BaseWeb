# M1-10 - 管理后台文章编辑页

> 对应任务: [01-迭代计划.md](../../plan/01-迭代计划.md#m1-10-管理后台文章编辑页markdown编辑器-预览)
> 涉及 Spec: [09-前端路由设计.md](../../spec/09-前端路由设计.md)

## 验证目标

管理后台文章编辑页支持 Markdown 编辑和实时预览。

## 测试用例

### TC-01: 编辑页访问

**前置条件：**
- 管理员已登录

**测试步骤：**
1. 访问 `/admin/posts/new`
2. 验证页面加载

**预期结果：**
- 编辑器显示
- 包含标题输入、内容编辑器、发布按钮

### TC-02: Markdown 编辑器

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 在编辑器输入 `# Hello World`
2. 验证内容显示

**预期结果：**
- Markdown 文本正常显示

### TC-03: 实时预览

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 在编辑器输入 `# Hello World`
2. 切换到预览模式或查看预览区

**预期结果：**
- 预览区显示渲染后的 HTML
- 标题样式正确

### TC-04: 保存草稿

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 填写标题和内容
2. 点击"保存草稿"按钮
3. 验证提示

**预期结果：**
- 保存成功
- 跳转到文章列表或留在当前页

### TC-05: 发布文章

**前置条件：**
- TC-04 已通过

**测试步骤：**
1. 点击"发布"按钮
2. 验证状态变更

**预期结果：**
- 发布成功
- 状态变为已发布

### TC-06: 编辑已有文章

**前置条件：**
- 存在已发布的文章

**测试步骤：**
1. 访问 `/admin/posts/{id}/edit`
2. 验证内容加载

**预期结果：**
- 标题和内容正确加载
- 可修改并保存
