# 12-05-标签云

## 12.5.1 功能概述

在博客列表页添加标签云组件，以可视化方式展示所有标签，标签字体大小反映文章数量。

## 12.5.2 页面布局

```
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│                   博客 / Blog                                │
│                                                              │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│                      标签云                                  │
│                                                              │
│     前端       Java        Vue        SQL        Git          │
│       Spring    React      API       Docker      Linux        │
│            TypeScript      CSS      MongoDB                    │
│                                                              │
├─────────────────────────────────────────────────────────────┤
│  筛选: [全部分类 ▼]  [全部标签 ▼]  [最新 ▼]                  │
└─────────────────────────────────────────────────────────────┘
```

## 12.5.3 功能详情

### 12.5.3.1 显示规则

|| 项目 | 说明 |
||------|------|
|| 字体大小计算 | `fontSize = 12 + (postCount / maxCount) * 16` |
|| 最小字体 | 12px |
|| 最大字体 | 28px |
|| 颜色配置 | 使用主题色或随机柔和色 |

### 12.5.3.2 交互行为

|| 交互 | 行为 |
||------|------|
|| 点击标签 | 跳转到 `/blog?tag={tagSlug}` |
|| 悬停 | 显示 tooltip："X 篇文章" |
|| 响应式 | 标签云在移动端简化为热门标签列表 |

### 12.5.3.3 组件设计

```vue
<TagCloud :tags="tags" @click="handleTagClick" />
```

|| 属性 | 类型 | 说明 |
||------|------|------|
|| tags | Array<{ id, name, slug, postCount }> | 标签列表 |

> 标签云组件的技术实现见 Spec: [12-05-标签云规格.md](../../spec/12-博客增强模块/12-05-标签云规格.md)
