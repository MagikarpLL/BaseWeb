|| 原则 | 说明 |

## 12.2.2 具体规格

### 12.2.2.1 后端接口（新增）

#### GET /api/blog/search

|| 接口 | 方法 | 说明 | 认证 |
||------|------|------|------|
|| `/api/blog/search` | GET | 搜索文章 | 否 |

**查询参数**：

|| 参数 | 类型 | 必填 | 说明 |
||------|------|------|------|
|| q | string | ✅ | 搜索关键词 |
|| page | int | ❌ | 页码，默认 1 |
|| size | int | ❌ | 每页数量，默认 10 |

**响应格式**：
```json
{
  "code": 200,
  "message": "Success",
  "data": {
    "posts": [
      {
        "id": 1,
        "title": "Vue3 实战教程",
        "slug": "vue3-tutorial",
        "excerpt": "本文介绍 Vue3 的使用方法...",
        "coverImage": "https://...",
        "categoryName": "前端",
        "publishedAt": "2026-05-20"
      }
    ],
    "pagination": {
      "page": 1,
      "size": 10,
      "total": 25,
      "totalPages": 3
    },
    "keyword": "Vue3"
  }
}
```

### 12.2.2.2 数据模型

**Service 层实现**（复用）：

```java
// BlogPostService.java
List<BlogPost> searchPosts(String keyword);
```

**SQL 模糊匹配**：
```sql
SELECT * FROM blog_post
WHERE status = 'published'
  AND (title ILIKE '%' || :keyword || '%'
       OR content ILIKE '%' || :keyword || '%')
ORDER BY published_at DESC
```

### 12.2.2.3 前端实现

#### 搜索组件

在 `BlogList.vue` 中修改：

```typescript
const searchQuery = ref('')

async function handleSearch() {
  if (!searchQuery.value.trim()) {
    fetchPosts()
    return
  }

  loading.value = true
  try {
    const res = await blogApi.search({
      q: searchQuery.value,
      page: currentPage.value,
      size: pageSize.value
    })
    posts.value = res.data.data.posts
    total.value = res.data.data.pagination.total
  } finally {
    loading.value = false
  }
}
```

#### 关键词高亮

```typescript
function highlightKeyword(text: string, keyword: string): string {
  if (!keyword) return text
  const regex = new RegExp(`(${keyword})`, 'gi')
  return text.replace(regex, '<mark class="highlight">$1</mark>')
}
```

**CSS 样式**：
```css
.mark.highlight {
  background-color: #fff3bf;
  padding: 0 2px;
  border-radius: 2px;
}
```
