# M1-02 - 前端项目初始化

> 对应任务: [01-迭代计划.md](../../plan/01-迭代计划.md#m1-02-前端项目初始化)
> 涉及 Spec: [01-技术架构.md](../../spec/01-技术架构.md), [06-02-前端项目结构.md](../../spec/06-项目规范/02-前端项目结构.md)

## 验证目标

前端项目可独立运行，包含 Vue 3 + Vite + 路由 + Pinia + 项目结构。

## 测试用例

### TC-01: 项目结构验证

**前置条件：**
- 无

**测试步骤：**
1. 检查 `frontend/` 目录存在
2. 检查 `package.json` 文件存在
3. 检查 `src/` 目录结构
4. 检查 `vite.config.ts` 存在

**预期结果：**
- 目录结构符合 `06-02-前端项目结构.md` 规范
- `package.json` 包含必要依赖（Vue 3, Vite, Vue Router, Pinia, Element Plus）

### TC-02: 依赖安装验证

**前置条件：**
- Node.js 18+ 已安装

**测试步骤：**
1. 进入 `frontend/` 目录
2. 执行 `npm install`

**预期结果：**
- 安装成功
- 生成 `node_modules/` 目录

### TC-03: 开发服务器验证

**前置条件：**
- TC-02 已通过

**测试步骤：**
1. 在 `frontend/` 目录执行 `npm run dev`
2. 等待服务启动
3. 访问 `http://localhost:5173`

**预期结果：**
- 服务启动成功
- 页面可访问

### TC-04: ESLint 验证

**前置条件：**
- TC-02 已通过

**测试步骤：**
1. 在 `frontend/` 目录执行 `npm run lint`

**预期结果：**
- 无 ESLint 错误
- 代码符合规范

### TC-05: 生产构建验证

**前置条件：**
- TC-02 已通过

**测试步骤：**
1. 在 `frontend/` 目录执行 `npm run build`

**预期结果：**
- 构建成功
- 生成 `dist/` 目录
