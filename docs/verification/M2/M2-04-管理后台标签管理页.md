# M2-04 - 管理后台标签管理页

> 对应任务: [01-迭代计划.md](../../plan/01-迭代计划.md#m2-04-管理后台标签管理页)
> 涉及 Spec: [09-前端路由设计.md](../../spec/09-前端路由设计.md)

## 验证目标

管理员可在后台管理标签，支持增删改查。

## 测试用例

### TC-01: 页面访问

**前置条件：**
- 管理员已登录

**测试步骤：**
1. 访问 `/admin/tags`
2. 验证页面加载

**预期结果：**
- 页面正常显示
- 左侧边栏标签管理菜单高亮

### TC-02: 标签列表显示

**前置条件：**
- 存在标签数据

**测试步骤：**
1. 访问 `/admin/tags`
2. 验证标签表格显示

**预期结果：**
- 表格显示 name、slug、操作列
- 操作列有编辑、删除按钮

### TC-03: 创建标签

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 点击"新增标签"按钮
2. 填写 name、slug
3. 点击保存

**预期结果：**
- 弹窗关闭
- 列表刷新，显示新标签

### TC-04: 编辑标签

**前置条件：**
- TC-03 已通过

**测试步骤：**
1. 点击标签行的编辑按钮
2. 修改 name
3. 点击保存

**预期结果：**
- 弹窗关闭
- 列表刷新，显示更新后的标签

### TC-05: 删除标签

**前置条件：**
- TC-03 已通过

**测试步骤：**
1. 点击标签行的删除按钮
2. 确认删除

**预期结果：**
- 列表刷新，标签消失

## 验收标准

- [ ] TC-01: 页面访问通过
- [ ] TC-02: 标签列表显示通过
- [ ] TC-03: 创建标签通过
- [ ] TC-04: 编辑标签通过
- [ ] TC-05: 删除标签通过
- [ ] 所有测试用例通过
