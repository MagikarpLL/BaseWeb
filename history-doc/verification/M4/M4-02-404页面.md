# M4-02 - 404 页面

> 对应任务: [01-迭代计划.md](../../plan/01-迭代计划.md#m4-02-404页面)
> 涉及 Spec: [09-前端路由设计.md](../../spec/09-前端路由设计.md)

## 验证目标

404 页面正常显示，提升用户体验。

## 测试用例

### TC-01: 404 页面访问

**前置条件：**
- 无

**测试步骤：**
1. 访问一个不存在的页面 `/not-exist-page-xxx`
2. 验证页面显示

**预期结果：**
- 返回 200（前端路由）
- 显示 404 页面内容
- 显示友好提示

### TC-02: 404 页面 SEO

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 访问 `/not-exist-page-xxx`
2. 检查 `<title>`

**预期结果：**
- `<title>` 包含 "404"

### TC-03: 返回首页链接

**前置条件：**
- TC-01 已通过

**测试步骤：**
1. 访问 404 页面
2. 点击返回首页

**预期结果：**
- 跳转回首页 `/`
