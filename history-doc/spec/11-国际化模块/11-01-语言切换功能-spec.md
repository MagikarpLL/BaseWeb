# 11-01-语言切换功能

> 对应 Requirement: [11-01-语言切换功能.md](../requirements/11-国际化模块/11-01-语言切换功能.md)

## 11.1 概述 / 设计原则

| 原则 | 说明 |
|------|------|
| 无刷新切换 | 语言切换局部生效，不触发页面刷新 |
| 按需加载 | 翻译文件按需加载，不影响首屏性能 |
| 持久化优先 | 用户偏好优先读取 localStorage，无则检测浏览器语言 |
| 渐进增强 | 基础翻译覆盖全部公开文本，SSR 兼容性预留 |

### 11.2.4 i18n 配置 (i18n/index.ts)

```typescript
import { createI18n } from 'vue-i18n'
import en from '@/locales/en.json'
import zhCN from '@/locales/zh-CN.json'

type MessageSchema = typeof en

const getDefaultLocale = (): 'en' | 'zh-CN' => {
  const stored = localStorage.getItem('locale')
  if (stored === 'en' || stored === 'zh-CN') return stored
  
  const browserLang = navigator.language || navigator.languages[0]
  if (browserLang.toLowerCase().startsWith('zh')) return 'zh-CN'
  return 'en'
}

const i18n = createI18n<[MessageSchema], 'en' | 'zh-CN'>({
  legacy: false,
  locale: getDefaultLocale(),
  fallbackLocale: 'en',
  messages: {
    'en': en,
    'zh-CN': zhCN
  }
})

export default i18n
```

### 11.2.5 useLocale composable

```typescript
// composables/useLocale.ts
import { ref, computed } from 'vue'
import { useI18n } from 'vue-i18n'

const LOCALE_KEY = 'locale'

export const useLocale = () => {
  const { locale, t } = useI18n()
  
  const currentLocale = computed(() => locale.value)
  const isZhCN = computed(() => locale.value === 'zh-CN')
  
  const setLocale = (newLocale: 'en' | 'zh-CN') => {
    locale.value = newLocale
    localStorage.setItem(LOCALE_KEY, newLocale)
    document.documentElement.lang = newLocale === 'zh-CN' ? 'zh-Hans' : 'en'
  }
  
  const toggleLocale = () => {
    setLocale(locale.value === 'en' ? 'zh-CN' : 'en')
  }
  
  return {
    locale: currentLocale,
    isZhCN,
    setLocale,
    toggleLocale,
    t
  }
}
```

### 11.2.6 LanguageSwitch 组件

**位置：** Header 导航栏，用户头像/登录按钮旁边

**功能：**
- 显示当前语言图标 (🌐 + 当前语言代码)
- 点击展开下拉选择 [English | 中文]
- 切换后立即生效，调用 `setLocale()`

**样式：** 与 Element Plus Dropdown 一致

### 11.2.7 Element Plus 中文配置

```typescript
// main.ts 或 i18n/index.ts 中配置
import ElementPlus from 'element-plus'
import zhCn from 'element-plus/dist/locale/zh-cn.mjs'

app.use(ElementPlus, { locale: zhCn })
```

**注意：** 管理后台强制使用 Element Plus 中文 locale，前台根据用户偏好动态切换。

### 11.2.8 管理后台语言设置

**路径：** Admin > Settings > Language

**功能：**
- 语言切换下拉框
- 保存后更新 localStorage 和 vue-i18n locale
- 页面顶部显示当前语言状态

### 11.2.9 关键逻辑

```
语言初始化流程：
1. 检查 localStorage['locale']
2. 如果存在，使用该值
3. 如果不存在，检测 navigator.language
4. 如果浏览器语言以 'zh' 开头，默认 'zh-CN'
5. 否则默认 'en'

语言切换流程：
1. 用户点击语言切换
2. 调用 setLocale(newLocale)
3. 更新 vue-i18n locale
4. 更新 localStorage
5. 更新 document.documentElement.lang
6. 不触发页面刷新
```
