# 语言切换功能需求

## 1. 需求概述

为网站添加全局多语言切换功能，支持中英文切换，用户偏好自动保存。

## 2. 用户场景

- 用户首次访问网站，语言跟随浏览器设置（Accept-Language）
- 用户可手动切换语言，切换后立即生效
- 用户偏好保存到 localStorage，刷新/关闭后仍保持
- 管理后台提供独立的语言设置入口

## 3. 功能需求

### 3.1 前台（Public）

| 模块 | 文本类型 | 需要翻译 |
|------|----------|----------|
| Header | 导航菜单 "Home", "Blog", "Tools", "About", "Login", "Logout", "Dashboard" | ✅ |
| Home | "Latest Posts", "Featured Tools", "View All", "Read Blog", "Explore Tools" | ✅ |
| About | "Skills", "Experience", "Projects", "View Project →" | ✅ |
| Tools | "Tools", "All", "Free online tools for developers" | ✅ |
| BlogList | 翻页、排序等控件文本 | ✅ |
| BlogDetail | 评论、分享等文本 | ✅ |
| Footer | "All rights reserved", "GitHub", "Contact" | ✅ |

### 3.2 管理后台（Admin）

| 模块 | 翻译范围 |
|------|----------|
| 导航菜单 | ✅ 全部翻译 ("Dashboard", "Posts", "Categories", "Tags", "Comments", "Settings", "Logout") |
| 表单标签 | ✅ 全部翻译 |
| 按钮文字 | ✅ 全部翻译 |
| 提示消息 | ✅ 全部翻译 |
| 页面标题 | ✅ 全部翻译 |

### 3.3 Element Plus 组件

- 全局配置 Element Plus 中文 locale
- 所有组件（ElButton, ElDialog, ElTable 等）自动使用中文

## 4. 技术实现

### 4.1 依赖

- vue-i18n@9

### 4.2 文件结构

```
frontend/src/
├── locales/
│   ├── en.json          # 英文翻译
│   └── zh-CN.json        # 中文翻译
├── i18n/
│   └── index.ts         # i18n 配置
├── composables/
│   └── useLocale.ts     # 语言切换 composable
└── components/
    └── LanguageSwitch.vue  # 语言切换组件
```

### 4.3 语言切换组件

**Header 位置**：在用户头像/登录旁边显示当前语言，点击下拉选择
**设置页面位置**：管理后台 - 设置 - 语言设置

### 4.4 持久化

- 语言偏好保存到 `localStorage` key: `locale`
- 默认值：`en`
- 检测浏览器语言：`navigator.language || navigator.languages[0]`

## 5. 非功能需求

- 语言切换无刷新重载，局部生效
- 翻译文件按需加载
- 兼容 SSR/SSG（如果未来需要）

## 6. 验收标准

1. ✅ 首次访问根据浏览器语言显示对应界面
2. ✅ Header 有语言切换下拉框，切换后立即生效
3. ✅ 管理后台设置页面可切换语言
4. ✅ 刷新/关闭浏览器后，语言偏好保持
5. ✅ 前台所有页面文本已翻译
6. ✅ 管理后台所有界面文本已翻译
7. ✅ Element Plus 组件显示中文
