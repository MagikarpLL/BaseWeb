# 04-01-用户认证

## 4.1.1 功能概述

用户认证模块负责管理员的登录、登出和令牌管理。

## 4.1.2 认证流程

```
┌─────────────────────────────────────────────────────────────┐
│                         登录流程                             │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│   用户                    前端                    后端        │
│    │                        │                        │      │
│    │  输入用户名/密码        │                        │      │
│    │ ──────────────────────► │                        │      │
│    │                        │ 提交登录请求           │      │
│    │                        │ ──────────────────────►│      │
│    │                        │                        │      │
│    │                        │     验证成功           │      │
│    │                        │ ◄──────────────────────│      │
│    │                        │   - Access Token (15min)│     │
│    │                        │   - Refresh Token (cookie)│    │
│    │                        │                        │      │
│    │  登录成功，跳转首页     │                        │      │
│    │ ◄────────────────────── │                        │      │
│    │                        │                        │      │
└─────────────────────────────────────────────────────────────┘
```

## 4.1.3 登录页面

### 页面布局

```
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│                      ┌─────────────────┐                    │
│                      │                 │                    │
│                      │   网站 Logo     │                    │
│                      │                 │                    │
│                      │   管理员登录    │                    │
│                      │                 │                    │
│                      │  用户名: [____] │                    │
│                      │                 │                    │
│                      │  密码:   [____] │                    │
│                      │                 │                    │
│                      │  [    登录    ] │                    │
│                      │                 │                    │
│                      └─────────────────┘                    │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 表单验证

| 字段 | 规则 | 错误提示 |
|------|------|----------|
| 用户名 | 必填，4-20 字符 | 请输入用户名 |
| 密码 | 必填，最少 6 字符 | 密码不能为空 |

## 4.1.4 密码错误次数限制

| 策略 | 值 |
|------|------|
| 5 分钟内最大错误次数 | 5 次 |
| 锁定时间 | 15 分钟 |

连续 5 次密码错误后，账户锁定 15 分钟。

## 4.1.5 页面权限控制

| 页面 | 访客 | 管理员 |
|------|------|--------|
| `/` | ✅ | ✅ |
| `/blog` | ✅ | ✅ |
| `/blog/:slug` | ✅ | ✅ |
| `/about` | ✅ | ✅ |
| `/tools` | ✅ | ✅ |
| `/login` | ✅ (登出后) | ❌ (重定向到/admin) |
| `/admin` | ❌ (重定向到/login) | ✅ |
| `/admin/*` | ❌ (重定向到/login) | ✅ |

## 4.1.6 验收标准

- [ ] 正确用户名密码可以登录成功
- [ ] 错误用户名密码返回错误提示
- [ ] 连续 5 次错误后账户锁定 15 分钟
- [ ] 登录成功后跳转管理后台
- [ ] Access Token 过期后自动刷新
- [ ] 刷新失败后跳转登录页
- [ ] 访问 `/admin` 未登录时重定向登录页
- [ ] 登录用户访问 `/login` 重定向到 `/admin`
- [ ] 登出后清除状态并跳转登录页
