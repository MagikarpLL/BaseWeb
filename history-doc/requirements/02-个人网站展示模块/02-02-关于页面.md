# 02-02-关于页面

## 2.2.1 功能概述

关于页面提供更详细的个人介绍，包括教育背景、工作经历、技能栈、项目展示等信息。

## 2.2.3 功能详情

### 个人介绍

| 字段 | 说明 | 格式 |
|------|------|------|
| 详细内容 | 个人详细介绍 | 富文本/Markdown |
| 位置 | 所在城市/国家 | 文本 |

### 技能展示

| 字段 | 说明 |
|------|------|
| 技能名称 | 如：Vue.js, Java, PostgreSQL |
| 熟练度 | 百分比或等级（精通/熟悉/了解） |
| 分类 | 前端/后端/数据库/工具 等 |

**展示方式**：
- 前端技能：进度条（0-100%）
- 后端技能：进度条
- 其他技能：标签云

### 经历（时间线）

**教育背景**：
| 字段 | 说明 |
|------|------|
| 时间范围 | 如：2015-2019 |
| 学校名称 | 文本 |
| 专业 | 文本 |
| 学位 | 本科/硕士/博士 |
| 描述 | 可选，详细描述 |

**工作经历**：
| 字段 | 说明 |
|------|------|
| 时间范围 | 如：2022-至今 |
| 公司名称 | 文本 |
| 职位 | 文本 |
| 描述 | 职责和成就 |

### 项目展示

| 字段 | 说明 |
|------|------|
| 项目名称 | 文本 |
| 项目描述 | 详细描述 |
| 技术栈 | 使用的技术 |
| 链接 | GitHub/在线链接（可选） |
| 图片 | 项目截图（可选） |
| 是否精选 | boolean |

**显示规则**：只显示精选项目，最多显示 6 个。

## 2.2.4 SEO 元数据

| 标签 | 内容 |
|------|------|
| title | `关于 - {姓名}` |
| description | `{姓名}的详细个人介绍，技能和工作经历` |
| og:title | `关于 - {姓名}` |
| og:type | profile |
